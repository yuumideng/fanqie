# 第十四篇 三千年悟剑 · 滚动状态档案

## 1. 人物境界线（当前值/篇末最终态）
- 秦云本尊：剑仙元神境，天仙极致境界（在雷兽府时间加速下修行约三千年达到）；本命飞剑已蜕变为"本命先天灵宝"（烟雨飞剑，内含一条完整大道，形成烟雨领域，威能比葵食宫主还强一筹）；两门上古妖族天帝大神通"周天星衣""周天星界"皆修炼到大成；实力达半步金仙层次（护身、领域为大能层次，杀敌手段为半步金仙层次）；自创困敌招数"江河滔滔"；剑仙天仙法门已有雏形；获雷兽府七枚先天雷果及一黄皮葫芦宝物；持千影暗光飞舟、百魔幡及十八魔仆等战利品；篇末准备去碧游宫万法池修行冲击金仙道果
- 伊萧：秦云之妻，龙族，居明耀大世界玉鼎门/大昌世界秦府
- 秦依依（秦云女儿）：人族与龙族混血，在明耀大世界闯荡，由敖邡龙王龙族照料
- 孟欢（秦云之子）：玉鼎门苍青峰弟子，元神三重天，道之领域六十里，修冰心剑道；本篇中得秦云指点剑术三月并授《烟雨》剑道书籍；欺压他的卢方等三名弟子被玉鼎门主惩戒
- 神霄道人张祖师：秦云好友，碧游宫弟子，天仙六重天（雷法境界天仙极致），自创神霄雷法；在雷兽府修行约一千余年，修成都天神雷阵圆满获十二杆阵旗，神霄雷法九重天法门已有五成把握；获三枚先天雷果及一葫芦
- 萨许师兄：碧游宫弟子，自创天罡雷法，媲美大能者实力；持三十六杆天罡阵旗（天罡雷法大阵）；在雷兽府闯过三座主殿，获七枚先天雷果及一葫芦；篇末惩戒夏侯道人夺其全部宝物
- 杨崧仙人：玉虚宫三代弟子，为报道侣之仇燃烧元神击杀寒鲨宫大宫主寒鲨将军，实力大降至普通天仙五六重天，修行路断绝
- 木龙护法：上古妖龙后皈依佛门，半步金仙，击杀寒鲨宫二宫主魇龙将军，因觊觎白玉令偷袭秦云未果后离去
- 夏侯道人：碧游宫弟子，半步金仙，擅护身手段；在雷兽府欲强夺张祖师法宝被秦云阻止；篇末被萨许、秦云联手困住，夺去全部宝物
- 葵食宫主：暗魔界两大宫主之一，半步祖魔实力，修肉身，擅近战，有尾巴绝招；两次刺杀秦云未果
- 魔尊（暗云魔尊）：明耀疆域魔道最高领袖，媲美金仙佛陀大能，派葵食宫主除秦云
- 火阐道人：玉虚宫三代弟子，天仙九重，大神通在身媲美金仙大能；强索白玉令未果与秦云结怨
- 罗宣（火德星君）：碧游宫金仙大能，天庭火部众神之首；夏侯道人求其出面被萨许拒绝
