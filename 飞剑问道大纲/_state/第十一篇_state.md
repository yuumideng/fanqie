# 第十一篇 灭蛮祖教 · 滚动状态档案

## 1. 人物境界线（当前值/篇末最终态）
- 秦云本尊：天仙境剑仙（凡俗之身），掌握如梦七剑全部七式（第七式"星如雨"于本篇第3章与妻子团聚时创出），道之领域三十里，本命飞剑一品（烟雨飞剑），拥乾坤环六枚（可构成"乾坤世界"困敌阵法）、下品灵宝"重水剑"（可分化百柄剑器施展周天剑光）、超品飞剑"鸳鸯飞剑"一对、青叶飞剑（超品法宝）、两界图图卷、上品灵宝"小番天印"（已献蒲曲龙君未收）。本篇末凝练剑仙元神失败，金丹碎裂、魂魄受创，约三个月后赴天狼界救女儿。获金光派掌门余通所赠《碧云仙法》（直通天仙九重天道家正统法门）及三册剑修一脉零星记载典籍。
- 伊萧：秦云之妻，神霄门真传弟子，龙族血脉觉醒达真龙境二重天，本篇第3章被秦云从蛮祖教六界绝天大阵救出团聚，法力封禁被秦云解除恢复真龙境气息，居于江州广凌秦府
- 秦云伊萧之女：出生即被贺谦（兀哈）送至魔神世界，后被黄袍尊者带走至天狼界，篇末确认仍活着，在天狼界一座山顶辛勤练剑（脸上有一道伤口），秦云决意赴天狼界营救
- 蒲曲龙君：祖龙境大能者，喜好收集宝物，本篇赐秦云大挪移道符（可施展两次）助其救妻，篇末应秦云所求推演女儿下落，查得在天狼界后因黄袍尊者缘故称"没法救"并合拢虚空缝隙，未收走小番天印
- 神霄道人张祖师：天仙三重天，道祖（灵宝天尊）亲传弟子，本篇揭露拥有"第二元神"（青袍中年道人），施展"雷罚"（黑白二气缠绕紫色雷霆）灭杀大批魔神护卫，自创神霄雷法故刻意停留天仙三重天未突破，被确立为大昌世界修行界领袖
- 白家老祖：上古天仙，拥上品灵宝"周天星辰"
- 摩诃菩萨：大昌世界佛门领袖，已修成金身
- 东海天龙：龙族天仙，篇末取走上古天龙霸昀骨灰安葬
- 天妖宫主（天妖娘娘）：九尾境界天妖，篇末参与灭蛮祖教大战
- 人皇：大昌世界人皇，真身镇压一处最大灾劫，本篇以化身参与大战
- 蛮祖教主（独角牛魔）：三重天巅峰魔神，修炼《大力牛魔体》至十七层，借蛮祖教"牛魔踏天阵"加持实力媲美天神天魔，被秦云"初见霹雳"重创心脏，篇末携猴老祖、羊角老魔借虚空挪移宝物逃离，踏上域外星空
- 猴老祖：妖魔九脉最擅长推演的古老存在，三重天巅峰魔神，篇末与蛮祖教主、羊角老魔分道扬镳，留在大昌世界遮掩天机隐匿
- 羊角老魔：三重天巅峰魔神，极擅长阵法，篇末随蛮祖教主踏上域外星空
- 鹫虎天魔：魔神帝君麾下天魔，转生降临大昌世界隐居蛮祖教，肉身比蛮祖教主更强，拥下品灵宝"双月魔刀"（两柄血色短刀），被秦云乾坤世界困住后获救，最终被张祖师"雷罚"灭杀化作飞灰
- 牛头魔神、狼头魔神：魔神帝君亲传弟子，仅有的两位能媲美天魔的三重天魔神，可越阶而战，可施展联手绝招（水火结合）。狼头魔神侥幸逃回魔神世界，牛头魔神被雷罚灭杀（剩晶莹骨骼，脏腑化飞灰）
- 魔神帝君（奎弗）：魔神世界帝君，两根弯角，实力远超大昌世界天仙天神，宫殿前有十六位强者雕塑（最弱为天魔五重天），企图征服大昌世界，因秦云破坏被迫提前发动世界通道阵法，篇末关闭崩溃世界通道
- 独角孩童：心魔一脉天魔，魔神帝君厚待的心腹，声音带迷惑力量
- 黄袍尊者：上古天庭时期大妖，灵宝道祖亲传弟子，西方白虎七星宿之首"奎木狼"，肉身成圣法门天妖九重天，极擅近战搏杀不亚于大能者，因妻儿皆亡性情大变反下天庭，独自居住天狼界，曾捉龙族研究血脉，与龙族关系极差。带走秦云伊萧之女
- 上古天龙霸昀：上古天龙宫之主，出生于龙山界，逃亡至大昌世界扎根，已死，遗愿是将骨灰撒回龙山界千转龙湖（被火傀老魔占据），篇末秦云将其尸体焚化骨灰交东海天龙
- 金光派掌门余通：明耀大世界金光派掌门，天仙八重天，灵宝一脉弟子（非道祖亲传），喊张祖师为"张师叔"，赠秦云《碧云仙法》及三册剑修典籍
- 洪九、朱八：秦云好友，转世仙人，见多识广
