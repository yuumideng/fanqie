# 第二十一篇 宝地 · 滚动状态档案

## 1. 人物境界线（当前值）

- **秦云**：散仙，金仙顶尖层次（元神法力+剑道结合），悟出天之大道、地之大道（皆顶尖大道），人之大道达瓶颈，离大道圆满只差最后一丝。本命飞剑"烟雨飞剑"经功德圆满质变为功德至宝（篇末达成）。持青萍剑（师尊灵宝天尊所借先天至宝，含圆满大道）。烟雨阵为顶尖金仙护身前三阵。本章渡劫准备就绪——获魂心果八颗、启灵果一颗（混沌源果已与孔宣交换）。
- **后羿**：大道圆满存在，三界攻击最强三位之一，箭术半步天道境，悟出"心力"一脉（心力第三境开创者，心力加持弓箭可越阶发挥天道境威力）。本篇燃烧元神连射九箭杀死博瀚岛主分身，心力接近力竭，百年内无法再爆发。
- **孔宣**：最接近半步天道境的几位之一，五色神光可夺宝（每日三次），领域为三界天道境下第一。本篇获两枚混沌源果、三枚魂心果。
- **木神句芒**：逼近半步天道境，擅藤蔓束缚擒拿。本篇获阵法圆盘（博瀚岛主遗物）、一枚混沌源果、三枚魂心果。
- **万花宫主**：红衣瘦弱男子，半步天道境阵法高手，擅长魅惑，持万花树、世界之须等宝物（被孔宣夺走）。本篇被秦云等擒拿后封禁实力，以带路前往幻魔山谷换取活命，篇中被放离去，收回万花宫。
- **博瀚岛主**：白发人，境界已达初始境（天道境），元神法力未突破，故意停留时空潮汐。兼修诸多异界修行体系，擅刀法（三柄弧形弯刀）、阵法（黑色圆盘）、护身甲铠（体表生长）。本篇分身被后羿燃烧元神第九箭射杀，本尊在博瀚岛未追击。
- **岩石大蛇**：幻魔山谷原生存在，半步天道境，擅时空阵法，在山谷内受天地力量滋养为不死之身。老主人为超脱时空潮汐的外来者，炼制幻魔山谷信物分散于时空潮汐。

## 3. 核心主线进度

- **时空潮汐探险线**：秦云、后羿、孔宣、木神句芒四人联手闯荡时空潮汐，先后击杀九幽船主人（心灵力量生命）、万花宫主（擒拿）、博瀚岛第三尊者摩沉（射杀），进入幻魔山谷获三大神果，最终击杀博瀚岛主分身后返回三界。——本篇完整收官
- **散仙渡劫准备线**：秦云为渡第十二次散仙之劫（致命之劫）搜集机缘，本篇获魂心果八颗、启灵果一颗，烟雨飞剑功德圆满质变为功德至宝。——渡劫在即
- **功德至宝诞生线**：秦云献祭三具博瀚岛阵法使者尸体，功德圆满，烟雨飞剑质变为功德至宝，可镇压一族一教气运。——本篇达成

## 4. 近章摘要队列（最近5章）

- 第15章：神果分配——后羿、秦云各一枚启灵果+一枚混沌源果+五枚魂心果；孔宣、句芒各一枚混沌源果+三枚魂心果。孔宣以三枚魂心果换秦云一颗混沌源果。博瀚岛主在山谷外等候。
- 第16章：博瀚岛主实力展现——境界已达初始境，以黑色圆盘布阵封禁、三柄弧形弯刀攻万花宫，后羿第九箭被挡，秦云燃烧元神撑烟雨阵。
- 第17章：后羿燃烧元神连射九箭，第九箭射杀博瀚岛主分身。博瀚岛主本尊在博瀚岛得知分身被杀，秦云等驾驭博瀚岛主飞舟全力逃回。
- 第18章：分宝——后羿得三柄弧形弯刀，秦云得十八面盾牌+飞舟，句芒得阵法圆盘，孔宣得银色丝线。放万花宫主离去。八十三年后抵达三界混沌时空缝隙。时空虫记下三界入口。
- 第19章（终）：秦云回雷啸山，献祭三具阵法使者尸体，功德圆满，烟雨飞剑质变为功德至宝，灵宝道祖感应喜称"可镇压一族一教气运"。
