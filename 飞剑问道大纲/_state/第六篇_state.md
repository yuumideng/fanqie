# 第六篇 玉瓶内的魔神 · 滚动状态档案

## 1. 人物境界线（当前值）
- 秦云：先天金丹境（紫金金丹），掌握守护、杀戮、黑白三大极境，自创烟雨剑诀（含广凌潮、江上明月、轮回等招），飞剑可感应百里、放出两千里；境界尚在极境，未入道。本篇新增装备：乾坤环（3件/一套灵宝的其中三件，单个一品法宝）、烟锁阵（混元宗所赠探查阵法，临时使用）、洞天剑葫（青铜葫芦法宝，超品层次，可灌百万剑气）、阿弥陀佛木雕（一梦百年，引领修行）、破旧石牌（神通雷霆之眼，仅余最后一次学习机会）。持有：七杀剑（一品，可化七柄二品）、本命飞剑（二品）、沉沙剑/盘尘剑/残血剑、火云葫（内含九转灵丹，本篇消耗两颗，余一颗）、天龙血晶（原得5颗，1颗换敖雪自由，余4颗）、上古天龙血。（域外魔神手臂本篇末换给张祖师换取洞天剑葫）
- 伊萧：先天实丹境，神霄门弟子，掌握天道意境/雷霆意境；与母敖雪在外行走。因秦云紫金金丹与自身龙族血脉，十年未有子嗣。
- 敖雪（龙女，伊萧之母）：纯血龙族，意境领域层次（战力媲美极境），恢复自由，与伊萧一同游历天下。
- 伊采石：伊萧之父，与武枫郡主在一起，与敖雪了断。
- 洪九：新晋先天金丹境，广凌郡两大先天金丹之一；其女与秦舒彦定亲。
- 秦舒彦（秦安之子，秦云侄儿）：十六岁，人剑合一，送入蜀州剑阁百剑峰修行。
- 秦舒冰（秦云侄女）：十六岁，未达人剑合一。

## 2. 伏笔追踪（已埋未解）
- [第六篇] 域外魔神"贺谦"——九山岛主献出大山真身后，域外魔神以紫光夺舍大山之体降临，自名"贺谦"，潜入人族世界，意图"占领这一方天地"——待回收（重大隐患）。
- [第六篇] 九山岛主夺舍"血海魔胎"肉身后逃离，魂魄气息未变，秦云一旦照面可识别——待回收。
- [第六篇] 玉瓶内魔神"卜明"被张祖师收为仆从，张祖师欲借其练成一门神通——待回收。
- [第六篇] 张祖师欲炼制天魔爪子套装灵宝，期待秦云入道后再取天魔另一条手臂——待回收。
- [第六篇] 西海龙族（敖充等）对秦云不满，暗中查探上古天龙宫新入口；深红鳞甲蛟龙"应蛟"亦在查探伊氏两名子弟——待回收。
- [第六篇] 秦云自创剑仙凝练元神法门之宏愿（张祖师点明只有道祖再传道或自创两途）——长线伏笔。
- [第六篇] 伊萧十年无子——待回收。
- [第六篇] 巡天盟/天下推测秦云"入道"（实为紫金金丹）——认知偏差伏笔。

## 3. 核心主线进度
- 主线A：秦云实力进阶与斩妖线 —— 紫金金丹法力大成，斩青翅妖王令天下大妖魔蛰伏；揭破于石奇魔神一脉身份并镇压玉瓶内域外魔神卜明，获张祖师赐宝。
- 主线B：岳母敖雪救出与家庭线 —— 以天龙血晶经万象殿换敖雪自由，母女相认；敖雪与伊采石了断。
- 主线C：玉瓶魔神线 —— 青翅妖王-于石奇-玉瓶-域外魔神卜明层层递进，终被秦云以"轮回"+九转灵丹镇压，归张祖师处置。
- 暗线D：域外魔神降临线 —— 九山岛主献身，魔神以"贺谦"之名潜入人世，为后续大患。

## 4. 近章摘要队列（最近5-10章一句话）
- 第21章：秦云以七杀剑周天剑光困住于石奇，乾坤环封禁其魔身，现出人族原形，殷离火认出其为玄武宫叛徒于石奇。
- 第22章：于石奇带秦云、殷离火至山腹刑窟，西门风、祁清水被阵法焚杀；于石奇坦陈因妻被夺之恨，临死欲告秦云瓶塞秘密。
- 第23章：于石奇话未说完被玉瓶灭口；秦云寻得紫蝉血木瓶塞，玉瓶惊恐下燃烧本源冲破封禁，青色魔神手臂自瓶口探出。
- 第24章：魔神卜明挣扎出瓶，秦云以广凌潮、江上明月、乾坤环围攻，魔神吞吸气血，秦云暴退拉开距离，紧急传讯伊氏老祖求援。
- 第25章：张祖师需盏茶时间赶到；秦云以绝招"轮回"连轰，将魔神砸回瓶中，服九转灵丹续力。
- 第26章：秦云连施八次轮回后再服一颗九转灵丹，终将魔神完全砸入玉瓶并塞上瓶塞，成功镇压，随即携瓶飞往神霄门。
- 第27章：张祖师途中以雷霆拦截汇合，带秦云回神霄门；紫袍青年"贺谦"掠过问茶山未发现踪迹后离去；张祖师识破秦云紫金金丹，论及剑仙凝练元神法门唯有道祖传道或自创。
- 第28章：张祖师拔开玉瓶认出域外魔神"卜明"，以"磨刑法"威慑令其臣服为仆；张祖师提出以五件宝物任选两件相酬。
- 第29章：张祖师展示五件宝物（黄巾力士、雷霆之眼石牌、青叶剑仙超品飞剑、洞天剑葫、阿弥陀木雕），秦云舍弃黄巾力士与青叶飞剑，选定阿弥陀木雕（一梦百年）与雷霆之眼石牌。
- 第30章：秦云另以域外魔神（天魔）手臂换得洞天剑葫；张祖师收卜明为仆并嘱保密；秦云归家告知伊萧，准备闭关"一梦百年"。本篇终。
