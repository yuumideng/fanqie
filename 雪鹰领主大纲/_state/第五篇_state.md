# 第五篇 薪火世界 · 滚动状态档案

## 1. 人物境界线（当前值）
- 东伯雪鹰：超凡/飞天级中期（身体），斗气飞天级初期；领悟万物之水、万物之火两种奥妙，火之奥妙已达万物境第二层次；掌握杀招"水火蛟龙杀"（水火奥妙融合）；修行《星火斗气》法门、《魔龙功》身体法门配套秘术《魔龙力》（第一重练成）；兵器为圣阶下品"黑龙枪"、"流冥靴"，护身内甲为雷真长老所留圣阶极品；太古血脉觉醒（力量爆发天赋，力量翻倍）。篇末名列夏族候补元老，入薪火宫"赤云山世界"修行。
- 池丘白（长风骑士）：圣级巅峰，半神实力，圣榜第一；掌握"空间切割真意"（三品真意）；安阳行省超凡领袖，创建长风学院。
- 彭山（妖刀骑士）：圣级，一千一百余岁，安阳行省年龄最大的超凡，创建妖刀院。
- 公良远：圣级（刚跨入不久），水源道观引路人，修行极品秘术"截风十五剑"（小成，能施展八剑）。
- 余靖秋：称号级法师（流星级），夏都城历练，对东伯雪鹰有情愫，立志跨入超凡。
- 晁青（副观主）：半神榜第九，水源道观副观主，掌握"雷神真意"，近两千八百六十余岁，接近寿命大限；夏族元老，提议东伯雪鹰列入候补元老。
- 司空阳（观主）：半神榜第五，水源道观观主，凝练"本尊神心"；夏族元老，赤云山修行师傅之一。
- 贺山主：天下第一强者（明面），夏族元老。
- 步城主：云雾城城主，半神中最美丽者，好强霸气；夏族元老。
- 叶玫（叶老太太）：半神榜第十，"杀神"，血刃酒馆大首领，天下第一杀手；与晁青打赌两次（第一次赌东伯雪鹰能否赢六场，输；第二次赌能否赢第八场，输）。

- 主线A：东伯雪鹰超凡成长线 —— 抵达水源道观领取资源→入夏都城→半年潜修创出"水火蛟龙杀"→超凡生死战赢九场一战成名→名列候补元老入赤云山修行。当前推进至：超凡生死战落幕，正式进入候补元老栽培阶段。
