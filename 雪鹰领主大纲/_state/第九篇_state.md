# 第九篇 划破长空 · 滚动状态档案

## 1. 人物境界线（篇末状态）

- **东伯雪鹰（本尊）**：圣级中期（身体修炼至不死之身），掌握三种二品真意（极点穿透、虚界真意、星辰真意）；身中"鬼六怨"巫毒，身体与斗气均无法修行，仅能靠真意和枪法修行（每天约半个时辰参悟，练枪可五个半时辰）；获"苦百回"解药一葫芦（够两百年服用），理论寿命两百年（实际可能不足百年）；获贺山主改造的空间碎裂宝物（克制空间封锁）；与余靖秋成婚，浪迹天涯。
- **东伯雪鹰（斗气分身）**：本尊遭巫神剑刺入时因灵魂剧痛导致斗气分身消散，后在薪火宫恢复（本尊清醒后可重新凝聚）。
- **余靖秋**：称号级法师（流星级），与东伯雪鹰成婚，陪东伯雪鹰浪迹天涯。
- **陈宫主**：薪火宫宫主，掌管薪火世界空间神器与丁九战船，主导围杀恶魔将军及攻打魔神会总部，承诺保护东伯雪鹰。
- **贺山主（贺源）**：天下第一强者，凝聚本尊神心，操纵丁九战船攻破魔神会总部法阵，为东伯雪鹰改造空间碎裂宝物。
- **晁青**：凝聚本尊神心（本篇突破），成为夏族第三位凝聚本尊神心者，实力名列半神榜前三；将进入红石山搏命。
- **司空阳**：水源道观观主，执掌"九龙火神甲"参加破阵救援。
- **太叔宫主**：海神宫宫主，执掌镇宫神器"海神王冠"和"镇海混元棍"，击杀寒冰恶魔（被三祭司救走）。
- **濮阳波**：被东伯雪鹰从魔神会总部救出，灵魂遭封禁折磨六年，经贺山主救治恢复。
- **谭石（左丘骑士）**：新晋半神，参加东伯雪鹰大婚。

## 2. 关键NPC状态

- **席云（大祭司）**：魔神会大祭司，夏族元老叛徒，被六尊神界战兵围杀毙命；灵魂可能被大魔神引领入黑暗深渊。
- **三祭司（暗红甲铠神秘人）**：魔神会最强半神，魔躯圆满，执掌两件神器（暗红甲铠+血色弯刀），救走寒冰恶魔拉尔夫，夺大魔神分身肉身失败逃跑。
- **奥兰大长老**：魔兽一族最高领袖，半神榜第三，巫神殿领袖，执掌巫神剑围杀东伯雪鹰后逃脱。
- **青魇王**：深海魔兽王者，半神榜第四，凝聚本尊神心，参与围杀东伯雪鹰。
- **蜘蛛女皇**：隐藏的魔兽一族强者，天赋足以名列半神榜前五，参与围杀东伯雪鹰。
- **拉尔夫（寒冰恶魔）**：五位恶魔将军之一，上位恶魔，被三祭司救走，臣服大魔神。
- **梅山主人（异界来客）**：落寞白发老者，实力境界高于夏族半神，以红石山情报为条件协助夏族破阵救东伯雪鹰。

## 4. 核心主线进度

- **恶魔将军围杀线**：东伯雪鹰六年探查找到五位恶魔将军 → 夏族倾巢出动围杀 → 四死一逃（拉尔夫被三祭司救走）→ 恶魔军团威胁基本解除。
- **魔神会总部攻破线**：东伯雪鹰发现魔神会总部 → 监视六年发现大魔神分身 → 三祭司抢夺分身失败 → 丁九战船封锁 → 东伯雪鹰夺走并摧毁大魔神分身肉身 → 六尊神界战兵杀大祭司席云 → 夏族半神攻破魔神会总部。
- **巫神大魔神阴谋线**：巫神与大魔神重新签订誓约，计划推迟五十年 → 针对东伯雪鹰的围杀 → 东伯雪鹰身中鬼六怨巫毒 → 巫神大魔神暂不再出手（代价太高）。
- **东伯雪鹰劫难与重生线**：遭围杀身中巫毒 → 夏族先辈紫雷帝君降世界投影救治 → 获苦百回解药（代价等同一件神器）→ 身体斗气无法修行但可练枪 → 与余靖秋成婚 → 浪迹天涯。

## 5. 近章摘要队列（最近10章）

- 第25章：东伯雪鹰靠不死之身和不死护体宝物硬抗巫神剑七剑，巫神之毒浓度达极致，夏族破阵到最后关头。
- 第26章：东伯雪鹰坠落昏迷，贺山主查探发现巫毒渗透粒子层面无法驱逐，梅山主人判断最多撑一个时辰。
- 第27章：丁九战船入薪火宫，陈宫主请夏族先辈神灵救东伯雪鹰，云海大帝指引找紫雷帝君，紫雷帝君降世界投影。
- 第28章：紫雷帝君辨认出"鬼六怨"巫毒，告知救治代价（一件神器）及后果（废人），夏族半神一致决定救。
- 第29章：东伯雪鹰醒来得知自身处境，身体斗气无法修行，获苦百回解药，见到余靖秋。
- 第30章：东伯雪鹰与余靖秋在赤云山世界试修行（仅半个时辰），余靖秋求婚，二人决定后天结婚。
- 第31章：消息传遍夏族，超凡们议论惋惜；巫神与大魔神确认东伯雪鹰不足为虑，暂不再出手。
- 第32章：东伯雪鹰第一天硬抗巫毒疼痛未昏迷，将魔神会宝物交还夏族，获空间碎裂宝物和五朵超凡七叶花，回雪石城堡练枪。
- 第33章：东伯雪鹰练枪五个半时辰（远超参悟真意的半个时辰），明悟应追求枪法而非真意，意外融合两种真意奥妙。
- 第34章：东伯雪鹰与余靖秋大婚，各方超凡来贺；晁青告别将入红石山；三月后二人浪迹天涯。
