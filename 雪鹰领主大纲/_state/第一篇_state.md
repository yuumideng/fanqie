# 第一篇 龙山楼黑铁令 · 滚动状态档案

## 1. 人物境界线（当前值）

- 东伯雪鹰：觉醒太古"力量血脉"；斗气达地阶骑士层次，实力远超地阶（常态流星骑士实力，力量爆发达银月骑士级）；枪法已达"枪法大师"境界，练成《玄冰枪法》第一层"飘雪"，枪速达"十二枪境"；装备飞雪神枪（二阶炼金兵器）、储物吊坠、护身内甲；位于雪鹰领雪石城堡。
- 东伯青石（弟弟）：5岁测精神力达标（赤红色光），具备法师天赋；即将拜白源之大法师为亲传弟子；位于雪鹰领。
- 东伯烈（父亲）：天阶骑士，被墨阳家族以谕令判罚"苦役百年"，已被带走。
- 墨阳瑜（母亲）：天阶法师，被墨阳家族以谕令判罚"禁闭百年"，已被带走。
- 宗凌（宗叔）：六臂蛇魔（蛇人族王族），天阶骑士，擅长六刀同时攻击与蛇尾身法；曾进入毁灭山脉外围修行；位于雪鹰领。
- 铜三（铜叔）：狮人（兽人族），天阶骑士，负责领地日常管理；位于雪鹰领。
- 盖斌（弯刀盟首领）：流星骑士（巅峰），已死于东伯雪鹰之手。
- 白源之：仪水城唯一星辰级法师（银月法师级），收青石为亲传弟子，位于仪水城。
- 司安：仪水城龙山楼楼主，前来送上黑铁令。

- [本篇第12章] 太古血脉觉醒者"初时一鸣惊人，之后却泯然众人，几乎都难跨入超凡"——东伯雪鹰能否打破此规律（待回收）。
- [本篇第24章] 弯刀盟老巢所在大山山壁浮现巨大岩石脸孔，感知到东伯雪鹰后悄然转移——超凡级存在的存在（重大伏笔，待回收）。

## 3. 核心主线进度

- 主线A：救父母线 —— 目标"得到龙山楼黑铁令"已达成，下一步获知墨阳家族详情（推进至篇末节点）。
- 主线B：修行成长线 —— 觉醒太古力量血脉→枪法大师→《玄冰枪法》第一层飘雪练成→灭弯刀盟、杀银月狼王、杀阴影豹（本篇内完成质变）。
- 主线C：弟弟青石法师路 —— 精神力达标→拜师白源之大法师（本篇完成铺垫）。
- 主线D：领地守护线 —— 灭弯刀盟除仪水城大患，声名鹊起，龙山楼关注。

## 4. 近章摘要队列（最近5-10章一句话）

- 第17章：进入毁灭山脉，首战黑鳞豺群，实战磨练枪法。
- 第18章：寻找弯刀盟老巢途中遇银月狼群放弃；五阶阴影豹偷袭雪鹰。
- 第19章：生死一刹那，雪鹰临死突破"力量圆满如一"，反杀阴影豹。
- 第20章：巩固枪法大师境界，练成《玄冰枪法》第一层"飘雪"，枪速达十二枪境。
- 第21章：灭银月狼群、杀银月狼王取心脏；弯刀盟探子发现雪鹰背狼王毛皮。
- 第22章：弯刀盟设伏峡谷，盖斌率近千匪徒围杀。
- 第23章：雪鹰短矛屠戮匪徒骨干，长枪刺杀盖斌，灭弯刀盟。
- 第24章：残余匪徒崩溃四散；弯刀盟老巢所在大山浮现神秘巨大脸孔（超凡存在），雪鹰感应逃离。
- 第25章：返回雪鹰领与弟弟团聚；龙山楼获悉弯刀盟被灭，楼主司安关注。
- 第26章：送银月之心给白源之，青石拜师成；龙山楼楼主司安亲至送上黑铁令（本篇终章）。
