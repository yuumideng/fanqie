# 第十三篇 决战 · 滚动状态档案

## 1. 人物境界线（当前值·篇末）

- 东伯雪鹰（本篇主角）：
  - 身体境界：万劫混元体第二层（《万劫混元经》第二层，体内每个粒子内部开辟小型虚界，粒子质变，身体翻天覆地提升）
  - 真意境界（本篇结尾）：星辰神心（二品神心境）、极点穿透神心（二品神心境）、虚界神心（二品神心境）——三门二品神心齐聚
  - 自创秘技：星辰陨灭击（融合星辰+极点穿透，威力大增）、混洞碾压（完全倾向力量，完善到极高地步）、大地（防御枪法，已完善）、凝真（虚界秘技，可消失无影无踪）
  - 界神级秘术：《万劫混元经》（第一层→第二层）、《大混洞真力》（第一层已入门，第二层太难练不成）
  - 核心装备：饮血枪（黑色长枪，神阶极品血炼神兵，枪尖泛金光）、星石火云枪、十二阶星塔（认主，可允许他人操纵）、青甲守卫（神界战兵）
  - 特殊能力：虚界映照整个夏族世界，可调动少许世界之力（未炼化世界之心）；一穿梭十万里（极点穿透神心）
  - 灵魂状态：损失一半灵魂本源（救妻子），成界神时间推迟十倍百倍，甚至可能成不了界神；但万劫混元体第二层灵魂受身体滋养强盛，损失一半不比战争初期弱
  - 实力评价：三种二品神心+一整套秘技+万劫混元体第二层+大混洞真力，可调动少许世界之力，在凡人世界无敌（一声怒喝灭杀三重天界神分身）
  - 当前所在：仪水城雪石山顶（篇末）

- 余靖秋（东伯雪鹰妻子）：半神法师；红尘岛琢磨星塔法阵八百年；强行全力操纵星塔近六年致灵魂本源耗尽至不足一成开始逸散；被大能者以东伯雪鹰一半灵魂本源转化之法救活，灵魂恢复苏醒；篇末在雪石山顶

- 晁青：已成神（神级初期），驾驭丁九战船；参与星塔决战、琢磨星塔法阵分担操纵压力

- 贺山主（贺源）：天下第一强者；与余靖秋轮换操纵星塔（一年十五万神晶目标），后期操纵时灵魂轻微受损

- 陈宫主（薪火宫宫主）：夏族领袖，协调各方；主导驱逐大地神殿信仰

- 司空阳（水源道观观主）：半神榜第五，参与战争

- 池丘白（长风骑士）：已死；实为魔神会三祭司、大魔神手下（掌握二品"大破灭真意"，修行分身法潜伏）；因不愿灭夏族反戈刺杀大魔神未遂，灵魂被大魔神抓走永远折磨（连同妻子"小希"），肉身留于星塔灵魂已逝

- 步城主、太叔宫主、宫愚、臧诺、淳圆：夏族半神，参与战争

- 袁青：实为魔兽一族天才（父母被人类所杀），此前透露消息致东伯雪鹰中巫毒；背叛时被巫神抢走送回其世界栽培

- 紫雷帝君：夏族先辈神灵，降临世界投影查看余靖秋伤势，帮忙联系时空神殿
