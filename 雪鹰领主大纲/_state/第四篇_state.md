# 第四篇 初识超凡 · 滚动状态档案

## 1. 人物境界线（当前值）
- 东伯雪鹰：超凡生命（飞天级），掌握万物之火、万物之水两种奥妙，水火关联互补；身体超凡化达飞天级中期（太古血脉一次觉醒，力量爆发型）；核心装备——圣阶极品护身内甲（雷真遗留）、圣阶下品长棍兵器（雷真遗留）、海洋界石（小型世界核心异宝，化灵液滋养身体灵魂）、源石三千余斤、金色猿猴与金色大鹏鸟两尊超凡炼金生物、黑色传讯手环（编号：水源安阳二九）；当前所在——雪石城堡（雪鹰领）
- 东伯雪鹰·斗气实丹：已超凡化，斗气完全转化为超凡斗气（火焰红色雾气）
- 东伯烈（父亲）：天阶骑士，二十年苦役后获救回归，世袭侯爵
- 墨阳瑜（母亲）：原流星级法师，因噩耗法力暴动致灵魂受伤成废人，经海洋界石灵液治愈恢复，白发（发根转黑），已回归雪石城堡
- 青石（弟弟）：青年，决意赴长风学院拜超凡法师为师修行法术
- 宗凌、铜三：兽人族，雪石城堡护卫长，长期守护东伯家族
- 池丘白（长风骑士）：水源道观长老，圣榜第一，圣级，掌握空间切割真意，安阳行省第一人，与东伯雪鹰结为兄弟（长风大哥）
- 谭石（左丘骑士）：海神宫长老，圣榜第六，圣级，掌握左丘真意，兽人族狮人
- 余靖秋：长风学院法师，28岁跨入称号级，已加入水源道观，将赴薪火世界
- 墨阳琦：墨阳家族族长，伪超凡（银袍女子），率家族向东伯雪鹰赔罪
- 墨阳琛：墨阳瑜之兄，新任墨阳家族族长
- 司良红：司家老祖，血妖之躯伪超凡，已率青河郡称号级向东伯雪鹰效忠
- 羿鸿：安阳行省龙山楼总楼主，伪超凡，为东伯雪鹰处理凡俗事务

## 3. 核心主线进度
- 主线A：东伯雪鹰黑风渊脱困修行线 —— 从坠入黑风渊到六年修行超凡化脱困回归，已完结
- 主线B：东伯家族团圆与清算线 —— 父母获救、墨阳家族赔罪、青河郡归东伯家族掌管，已推进至收束阶段
- 主线C：超凡组织选择线 —— 六大超凡组织纷纷邀请，东伯雪鹰选定水源道观，已定
- 主线D：东伯雪鹰超凡修行路 —— 即将赴水源道观、薪火世界修行，开启新阶段

## 4. 近章摘要队列（最近10章）
- 第19章：东伯雪鹰至东香湖炼金作坊救父，超凡威压震慑墨阳家族看守，向墨阳家族下最后通牒
- 第20章：墨阳家族族长墨阳琦下令肃清内部，墨阳辰白被处死；东伯雪鹰携父母归雪石城堡，龙山楼羿鸿楼主来访
- 第21章：羿鸿向东伯雪鹰介绍圣榜（三十位）与半神榜（十位），池丘白名列圣榜第一，巫神殿大长老为人类最可怕敌人
- 第22章：羿鸿介绍六大超凡组织（大地神殿、血刃酒馆、黑白神山、云雾城、水源道观、海神宫）及魔神会；深夜黑夜刀魔涂亮（圣榜第十）来访试探交手
- 第23章：涂亮与东伯雪鹰交手，水神骑士温梁（圣榜第二十一）到来，两大圣榜高手各递招揽条件
- 第24章：池丘白、谭石（圣榜第六）亲至，水源道观与海神宫各递条件；云雾城、黑白神山亦派人至，六大组织尽皆邀请
- 第25章：东伯雪鹰选定加入水源道观，池丘白赠传讯手环（编号水源安阳二九），讲解传讯规则与超凡世界冒险风险
- 第26章：龙山楼羿鸿接手清整青河郡，司家势力被连根拔起；东伯雪鹰向家人展示海洋界石灵液并叮嘱保密
- 第27章：青河郡称号级司良红等六人前来效忠东伯雪鹰；青石决意赴长风学院拜超凡法师为师
- 第28章：墨阳琦率墨阳家族长老至雪石城堡跪地赔罪，交卷宗（处死5869人、判罪两万余人）、一千万金票、推举墨阳琛任新族长；东伯雪鹰出发前往水源道观

## 5. 待确认/存疑
- 第3章中描述裂缝"从地底深处劈出，贯穿上百里"——原文第9章雷真记载为"行进足足一千多里"，第3章东伯雪鹰观察裂缝"二三十里长"为地面可见部分；第4章又描述"地底几乎都是山石泥土堆积，一击贯穿上百里"。原文此处"上百里"与"一千多里"并存，推测地底裂缝总长一千多里、地面残余裂缝二三十里、一击贯穿的直线破坏范围上百里，三者不矛盾，留此备注。
- 第21章半神榜"半神寿命三千年"与第22章"半神寿命也就三千年"一致，无矛盾。
