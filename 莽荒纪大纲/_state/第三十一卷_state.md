# 第三十一卷 星河流域 · 滚动状态档案

## 1. 人物境界线（当前值）

- 纪宁本尊：二步道君，终极剑道第二层（五大剑意中滴血、无影、阴阳已达第三层，唯心、天崩卡在第三层瓶颈），心力修行者（心力第六层"世界"，心之世界媲美一座疆域，达合道边缘层次）；核心装备：主宰甲衣、六柄北虹剑（金色蜕变，滴血式超越空间）、九大秘术、世界投影、幻星术（自创心力幻术）、心力灭绝术（羽裳道君所传改良）；当前所在：易波界外（大莫域），与九尘教主一同进入易波界
- 纪宁法身：法身实力约本尊七成（无北虹剑），坐镇天苍宫；已展露心力修行者身份，"北冥道君"之名传遍无尽疆域
- 第二元神：开辟心之世界，实力强大；保管悟道之宝"离合石府香炉"（三界内）
- 九尘教主（赤水）：古修行者，四步道君（本卷在霆南行宫突破），合道边缘，"水之涟漪"秘术大成，水行不死身；持宇宙之宝暗红色长棍、掌心阵图法宝（先祖赐予）、寰裂石；当前与纪宁一同进入易波界
- 天一道君：三步道君，合道边缘，法身在易波界内（与大莫道君法身一同），本尊所在不明；曾与大莫道君法身一同探索易波界，遭遇哥雀王险些丧命
- 大莫道君：三步道君，推演一道极强（战斗力较弱，与天一、清风圣主相当），本尊带妻子躲藏，仅留法身坐镇大莫院
- 来琊道君：星河一族，合道边缘，本卷在霆南行宫合道失败（永恒之花仅花骨朵即枯萎），魂魄真灵开始溃散，决定离开虞星海去无尽疆域游历
- 风霄公子：纪宁三徒弟，留在家乡星辰圣地修行

## 2. 伏笔追踪（已埋未解）

- [本卷第1章] 阴阳生死轮——虞星海中近百疆域大小的双漩涡（生轮汲取生机、死轮吞噬一切），不断扩张变大，约一百多万混沌纪后将毁灭无尽疆域；三大主宰都无法伤害它——待回收的终极悬念
- [本卷第5章] 幻尘文字——幻尘道异兽首领眉心独眼上的奇异文字（七十二道神纹构成），蕴含极可怕的幻术奥妙，纪宁已记下并钻研——待深入参悟
- [本卷第10章] 幻星术——纪宁融合浮木桥幻术、幻尘文字、羽裳道君《幻界》经验自创的心力幻术，尚需完善提升——待持续提升
- [本卷第11-12章] 离合石府香炉——霆南行宫悟道之宝，纪宁合道最重要的宝物，已放在第二元神处保管——待长期使用
- [本卷第10章] 雷霆守护者透露——霆南行宫主人在上古之战前留下四座行宫分散在虞星海各处，通过四处考验可有大机缘——待回收（但虞星海危险难寻）
- [本卷第15章] 洞溟玉符——合道至宝，易波帝君炼制六份（道盟一份，五份留易波界），可形成虚影分身尝试合道，失败对本尊无伤；本卷出世引发无尽疆域道君疯狂——待争夺回收
- [本卷第16章] 各方势力合道边缘道君齐聚易波界——哥雀王（永恒一族）、石九（芒涯国皇族九王）、天斧道君、夫一道君、黎星宫主、光明王、酒圣（黑暗国度，无尽疆域公认第一道君，持宇宙之宝）等——待易波界大战
- [本卷第24章] 酒圣获得宇宙之宝——黑暗国度第一道君酒圣得到宇宙之宝，强行杀破道盟万道君大阵进入易波界——待展开

## 3. 核心主线进度

- 主线A：虞星海归途线 —— 已完成。纪宁与九尘教主从虞星海极深处出发，经阴阳生死轮、幻尘道、星河流域、霆南行宫，最终安全返回无尽疆域
- 主线B：修行突破线 —— 进行中。纪宁心力修行者身份逐步展露；五大剑意中三道达第三层，唯心、天崩卡在瓶颈；九尘教主突破四步道君；来琊道君合道失败
- 主线C：洞溟玉符争夺线 —— 刚开启。易波界洞溟玉符出世，各方云集，纪宁与九尘教主进入易波界
