# 第三十六卷 四步道君 · 滚动状态档案

## 1. 人物境界线（当前值）
- 纪宁本尊：四步道君（终极剑道第四层，道树十万八千丈）。终极剑道五式（滴血、无影、天崩、阴阳、唯心）均已融合至第四层次；心剑术第十二式"剑如潮"（六倍增幅）；修成不死身"无影之躯"（我为天地，天地为我）；主宰级秘术《分光大剑气》前两重练成、《神翼剑光》前两层练成，二者以终极剑道之阴阳结合形成"阴阳混沌世界"；心力幻术《大自在梦界》创成（圣城之主级数）。实力已达圣城之主巅峰，与道盟两大领袖相当，三大主宰之下近乎无敌。本卷末离开青花洞府前往虞星海瑶火境救援九尘。
- 纪宁第二元神（黑色道袍纪宁）：留守，本卷末接替本尊在青花洞府至尊石台上继续修炼创法门。
- 纪宁法身（金衣纪宁）：驻天苍宫，本卷末与苏尤姬、天一帝君饮酒时得知九尘陷入瑶火境。
- 九尘教主（赤水道君）：四步道君，水行不死身；本卷初与纪宁在天苍宫分离独行合道之路，后独入虞星海冒险，被绯雪城主追杀逃入绝地"瑶火境"。
- 绯雪城主：八大圣城之主前三，刀道+冰雪双道合道，不死身（雪花遁）；本卷被纪宁击败于绯雪圣城后逃遁，藏匿于无尽疆域边缘修复残缺域界飞舟，后入虞星海追踪九尘，设"瑶火境"陷阱意图引纪宁入绝地借刀杀人。
- 血云帝君：实力与绯雪城主相当，本卷绯雪圣城一役被纪宁击败后撕裂时空逃遁。
- 芒涯主宰：三大主宰之首（混沌神兽本尊），本卷暗中观战绯雪圣城一役，震撼于纪宁实力；主宰炎龙域界大会，为纪宁站台。
- 青石道人：道盟第一帝君（非主宰却媲美主宰），本尊修黑白之道，第二元神以舍身仪式转化为西斯族"黑衣帝君"（主宰级战力）；本卷以超过九成宝物为代价换取纪宁《终极剑道》一观，与纪宁结为好友，赠予西斯护卫"白融"、二十五滴银衣道君之躯、葫芦宝物及虞星海详细星图。
- 天一帝君：天苍宫之主，擅不死身保命手段，纪宁结义大哥。
- 苏尤姬、丹宝：纪宁仆从（天苍宫）。
- 白融：西斯族傀儡护卫（圣城之主巅峰，擅长保护，可纠缠主宰一二），青石道人赠予纪宁。
- 四大守护傀儡（持巨斧/短矛等）：圣城之主级数，原魔象石壁西斯遗迹守护者，本卷被纪宁擒拿炼化认主，后安置于三界。
- 启至尊（已逝）：青花洞府府主，至尊，道号"启"，已死（"算是自杀"），留下青花洞府三重考验、两大文明无数法门、十二卷终极之道、至尊石台及遗愿（创第十重混沌禁制）。
- 洞府之灵（白发老者）：青花洞府守候者，引导纪宁闯关及修炼。

## 2. 伏笔追踪（已埋未解）
- [本卷第1章] 纪宁本尊借道符复活后护体神通需重新修炼——《剑道之身》第四层（神体媲美永恒极品法宝）所需珍材极多，本卷以赤波殿果实向芒涯主宰换取珍材并修炼完成（第1章交代计划，后续未明确描写完成时点）。
- [本卷第3-4章] 纪宁在天苍永恒界凡俗大地一亿三千万年悟道、化身顽石数百万年——终极剑道突破契机，第4章已突破四步道君，已回收。
- [本卷第6章] 神兵谷断剑宇宙之宝（法宝之灵冰寒女子）拒绝跟随纪宁，要求纪宁合道成功方才答应——待回收。
- [本卷第14章] 绯雪圣城一役震动无尽疆域，吸引诸多隐居帝君老家伙现身——部分已回收（炎龙域界大会），其余隐居者去向待展开。
- [本卷第16章] 青石道人第二元神"黑衣帝君"在外冒险——其具体行踪及与纪宁后续互动待展开。
- [本卷第17章] 纪宁公开"融合最强之道之上还有终极之道"这一秘密，并承诺合道失败身死道消前可请青石道人出手——待回收。
- [本卷第21章] 绯雪城主持有残缺域界飞舟（甚至飞舟之灵完好、可使用）这一最大秘密从未公开，三大主宰若知将全力斩杀——重大未解伏笔。
- [本卷第22章] 魔象石壁西斯遗迹"虎战船"战争区有十枚"始龙石"（一颗始龙石可换一尊黑衣帝君），疑似已被波林至尊取走——待确认。
- [本卷第23-27章] 青花洞府府主"启至尊"已死，死因"算是自杀"（并非真正自杀），具体死因未明——重大未解伏笔。
- [本卷第27章] 启至尊搜集十二卷终极之道（十二名合道失败的终极之道道君），终极之道合道极难——纪宁合道悬念。
- [本卷第28章] 启至尊遗愿：创出九重混沌禁制之上的第十重（转化永恒帝君神力法力）、乃至第十一重、第十二重（转化至尊神力法力）；纪宁需先创出十种圣城之主级数法门方可获赠一件"对合道帮助排第一"的宝物——核心待回收。
- [本卷第29章] 纪宁已创两门圣城之主级数法门（终极剑道、大自在梦界），尚需创八门——进行中。
- [本卷第30章] 九尘陷入虞星海瑶火境，绯雪城主设陷阱引纪宁入绝地借刀杀人——下卷核心冲突。
- [本卷第30章] 天一帝君好友"大莫道君"擅推演一道，提醒此事针对纪宁——大莫道君身份及推演能力待展开。
- [上卷遗留] 纪宁拔走的变异万劫圣血果树（三十六枚永恒之果）——本卷已用于与芒涯主宰交易珍材（第7章），部分永恒之果仍在。
- [上卷遗留] 永恒一族三位帝君"不死不休"追杀令——本卷安臣帝君法身、岐隐帝君仍参与追查，但被纪宁实力震慑，未正面交锋。

## 3. 核心主线进度
- 主线A：纪宁成就四步道君并展露锋芒——已完成（第4章突破，第10-15章绯雪圣城一役震慑四方，第16-19章炎龙域界大会确立地位）。
- 主线B：纪宁积累宝物以求请至尊复活师姐余薇——持续推进（青石道人赠宝、魔象石壁扫荡、青花洞府机缘，尚不足请至尊）。
- 主线C：纪宁悟道与创法门（终极剑道突破、秘术修炼、青花洞府创十种法门）——进行中（已创两门，尚需八门）。
- 主线D：绯雪城主追杀与反杀——本卷纪宁反杀挫败，绯雪城主逃遁设新陷阱（瑶火境），冲突延续至下卷。
- 主线E：九尘独行合道之路——本卷分离，末尾陷入瑶火境危机，下卷救援。

## 4. 近章摘要队列（最近10章）
- 第21章：纪宁再入魔象石壁，破弗魔族禁制收走帝级/皇级傀儡及法门，进入西斯遗迹。
- 第22章：纪宁擒下四大守护傀儡，得八根魔雀晶及八十六尊帝级傀儡，安置于三界。
- 第23章：纪宁再入青花洞府，一掌击败血眸老者，洞府之灵现身告知三重生死关及终极之道。
- 第24章：纪宁通过第二重关（击败十名帝君、推开殿门），获赠一卷终极之道（终极火焰道）玉简。
- 第25章：纪宁通过第三重关（将圣城之主级黑袍瘦长男子轰出大殿使其被禁制击杀）。
- 第26章：纪宁进入第一殿厅，观看两大文明（修行者+西斯族改造）无数法门（超千亿卷玉简），得知府主为"启至尊"且已死，磕头感恩。
- 第27章：纪宁学十一卷终极之道及启至尊神念传承（道韵），进入第二殿厅见至尊石台。
- 第28章：洞府之灵揭示启至尊遗愿——创第十重混沌禁制；要求纪宁先创十种圣城之主级数法门方可获赠"对合道帮助排第一"的宝物。
- 第29章：纪宁创出第二门法门《大自在梦界》（心力幻术，圣城之主级），真实耗时大半个混沌纪；继续钻研水行一道。
- 第30章（终章）：纪宁真实修炼已超一个混沌纪；得知九尘陷入虞星海瑶火境（绯雪城主陷阱），本尊离青花洞府赴虞星海，第二元神留石台继续修炼。

## 5. 待确认/存疑
- 【存疑1】第1章纪宁以赤波殿果实向芒涯主宰换取《剑道之身》第四层珍材，芒涯主宰承诺一月内送达；后续章节未明确描写纪宁修炼《剑道之身》第四层完成的时点，仅第4章提及"护体神通需重新修炼"。按原文，珍材交易完成（第2章"大半个月后芒涯国有道君前来送宝物"），但护体神通修炼完成时点原文未明写，存疑。
- 【存疑2】第4章纪宁突破四步道君后"心剑术如今已经练成了十二式"，第29章回溯"在凡俗中行走一亿三千万年时就自然而然突破到了新的层次"（指心剑术第十二式）。两处一致，无矛盾。
- 【存疑3】第16章炎龙域界大会出席大能为"九位"：芒涯主宰、风雨主宰、冥兰主宰、黑云帝君、金屿帝君、金孔帝君、凰琼帝君、青石道人，加上纪宁共九位。原文明确"一共足足九位大能"，与金屿帝君介绍的"道盟五位帝君+三大主宰"（金屿、黑云、金孔、凰琼、青石=5；芒涯、风雨、冥兰=3）合计8位+纪宁=9位，一致。
- 【存疑4】第30章纪宁真实修炼时间"已经超过一个混沌纪"（含石台百倍加速下实际过百亿年，真实时间大半个混沌纪创大自在梦界后继续钻研水行一道）。原文时间表述："转眼在青花洞府中，纪宁修行真实度过的时间已经超过一个混沌纪了"。一个混沌纪的具体换算（混沌纪≈？万年）原文未给精确数值，按莽荒纪体系，一个混沌纪极漫长，此处忠于原文表述。
- 【存疑5】第22章"始龙石"——原文述"一颗始龙石能抵十枚魔雀晶""战争区据说有足足十枚始龙石""始龙石估计早就被抢走了"。始龙石是否确实被波林至尊取走，原文仅为傀儡猜测（"据我们猜测"），未确证，存疑。
