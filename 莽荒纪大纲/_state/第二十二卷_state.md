# 第二十二卷 真神 · 滚动状态档案

## 1. 人物境界线（卷末）

- 纪宁本尊：突破至真神（半步祖神，以剑道为核心重塑神体）；神魔炼体《赤明九天图》第十六重；八九玄功第九转；摘星手第五转；剑力达第五层次"剑神境"，《伍宝剑术》大成，剑速超越天道极限；心力第四层次；烛龙之眼第二重本命烛光；掌握《枯寂世界神》（已修炼，神体经《金像》神通上部淬炼至媲美先天极品法宝）、《九元灭》神通、《唯一本尊》秘术、《心沁之法》；护体神通《金像》神通上部（神体媲美先天极品法宝）；自创《明月剑术》（天崩式、滴血式、无影式）；习得《剑天地》七剑式（剑开屏、天剑坠等，北休世界神碑石剑术）；炼化神剑"紫光琼"表层修复（可发挥部分威能，进攻达顶尖祖神层次）；六柄北冥剑均达混沌奇宝普品层次；持窥天太皓塔（混沌奇宝，时间流速最高千倍，维持百倍修炼）；获释狱真神三件混沌奇宝（含暝风裂空棍换得的一对棍、窥天太皓塔）；身处上古废墟/三界万族殿
- 纪宁第二元神：在斜月大世界窥天太皓塔中修炼悟道（百倍时间流速，外界十九年≈塔内近两千年），剑术持续精进；另有一化身陪女儿明月
- 余薇（纪宁妻子/师姐）：真灵已被黑袍神王捏碎（逐鹿大世界一战），尚未恢复
- 佛祖如来：本尊释迦摩尼法身重伤濒死（神力耗尽，仅余金身），大日如来法身、药师如来法身被刺修大魔神击散消解（两大法身葬送），仅余本尊金身
- 菩提道祖（纪宁师傅）：坐镇三界万族殿，关注纪宁安危
- 女娲娘娘：已离开三界进入无尽混沌（成世界神后离去）
- 伏羲氏：人皇，真神道祖（转世未恢复祖神记忆），阵法三界第一，领悟阴阳两条天道
- 后羿、燧人氏：领袖层次大能，率三十六名真神道祖驰援上古废墟

## 2. 关键法宝

- 紫光琼神剑：表层已修复（耗费近百件先天灵宝+一尊万物使者采五行之精），内部仍千疮百孔，仅能发挥部分威能；动用时进攻达顶尖祖神层次
- 窥天太皓塔：释狱真神所获混沌奇宝，金字塔状，可改变时空流速（最高千倍），纪宁炼化后用于第二元神悟道修炼
- 北冥剑六柄：均达混沌奇宝普品层次
- 五行神鼎：先天层次，可毁先天灵宝采集五行之精
- 万物使者（傀儡）：已损坏，投入五行神鼎毁掉用于修复紫光琼表层
- 如意灵蛇梭：先天灵宝，遁逃飞行之用
- 接引令牌：天苍宫接引信物（成祖神后可用）
- 大千剑阵：七百二十九口纯阳极品飞剑
