# 第四十一卷 护道楼 · 滚动状态档案

## 1. 人物境界线（当前值）

- **纪宁本尊**：永恒终极剑道帝君（合道失败，真灵时刻溃散）；本卷起始寿命约一万两千混沌纪，本卷末因狱界战斗全力出手寿命锐减至六百混沌纪（全力出剑百次左右即身死）；掌握剑道本源领域（媲美异宇宙之主全力一击，纯凭领域可灭杀主宰）；本卷新悟成空间剑道、风之剑道、风雷剑道（风之剑道与雷电剑道融合）；持北虹剑（六柄金色神剑）、青花洞府；携带域界飞舟
- **纪宁第二元神**：黑色道袍，本卷中在外界与鸿然至尊法身、貘谷至尊法身一同寻找破解隐秘空间之法；维持百倍时间加速辅助本尊参悟
- **丹尊者**：主宰境（剑道主宰之妹/挚爱），黑纱清冷女子，为复活三位兄长长期疯魔深入西斯族族地寻宝，持数件西斯族兵器；本卷获赠一艘域界飞舟；三位兄长（大哥剑道主宰、二哥白衣男子帝君、三哥青甲男子帝君）本卷由天食至尊逆转时空复活
- **青魔主宰**：纪宁追随者，本尊留守保护家人，法身随纪宁冒险西斯族族地；本卷获赠红铜色甲铠西斯族兵器、黑色锁链兵器（暂时使用）
- **毁灭神庭之主**：异宇宙之主（主宰与异宇宙之主之间实力），化身初登场求纪宁救丹尊者；本尊、法身均存活，法身在本卷参与异宇宙内复活三位帝君
- **缇雅主宰**：银发灰袍女子，"疯子"，长期在西斯族族地冒险，平静淡然心已死（挚爱不可复活）
- **炎左主宰**：黑发独角男子，被西斯族囚禁折磨漫长岁月未屈服，本卷被纪宁救出，获赠臂环西斯族兵器
- **芒涯主宰**：雪白衣袍白胡子老者，引荐毁灭神庭之主见纪宁
- **天食至尊**：万道时间本源，本尊亲自出手在异宇宙逆转时空复活丹尊者三位兄长
- **鸿然至尊**：万道因果本源，法身（红袍秃顶老者）参与寻找隐秘空间
- **貘谷至尊**：万道空间本源，镇守西斯族族地，法身（高瘦青袍鳞片男子）参与寻找隐秘空间，接纪宁第二元神
- **波万月（西斯族尊主）**：深蓝长袍青年，深蓝长发，隐秘空间留守尊主，掌控祖殿，欲杀纪宁立功

## 3. 核心主线进度

- **救丹尊者线**：已完成。纪宁应毁灭神庭之主所托，闯入西斯族族地护道楼所在，破空间走廊锁救出丹尊者及同伴缇雅、毁灭神庭之主，并救出被囚禁的炎左主宰
- **复活三位帝君线**：已完成。纪宁请天食至尊出手，在异宇宙古域起源之地内域逆转时空复活丹尊者三位兄长（剑道主宰及两位帝君）
- **护道楼之战线**：已完成。纪宁破开护道楼所在，护道楼自毁放出奇异生命，纪宁六剑灭杀奇异生命，拆卸护道楼获二十一件西斯族兵器及矮小生物洞天法宝内二十九件宝物（含两艘域界飞舟、求败神主灭绝矛）
- **隐秘空间困局线**：进行中。纪宁一行被九空间大阵挪移入隐秘空间（六普通世界+圣界+狱界），至尊法身无法锁定位置；纪宁闯狱界逐个击破被放逐者，破堡垒、破时空大阵，寿命锐减至六百混沌纪；即将闯圣界
- **九种特殊法门线**：新埋伏笔。纪宁从西斯族后裔记忆中发现九种特殊修行法门，欲寻找以创完美法门续命
