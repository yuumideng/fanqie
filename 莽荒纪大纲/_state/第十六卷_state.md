# 第十六卷 寂灭之域 · 滚动状态档案

## 1. 人物境界线（卷末）

- 纪宁本尊：返虚前期地仙；神魔炼体《赤明九天图》第十七重（渡劫前极致）；八九玄功第五转；摘星手第四转；剑道未大成但掌握一丝"剑之力"（剑道本质追寻所得）；心力达第三层次"主宰"（《后羿箭术》心力五境界之第三，三界顶尖神箭手水准）；悟出心力在剑指上的运用法门（一招消耗约十五分之一心力）；业火缠身（已扛过灭魂劫、红尘劫、千世心劫三重劫难，虚无业火永伴灼烧神魂）；从寂灭之域被菩提老祖救回归大夏世界
- 纪宁第二元神：返虚后期地仙；修炼《北冥经》；准备近期渡劫成天仙；菩提老祖嘱其悟出心力用在仙剑上的办法后再渡劫
- 余薇：已渡天劫成为天仙（七九雷劫+心魔劫）；前世为修罗存在实力极强；心魔劫中与纪宁在心魔世界共度三千年；回归大夏世界黑白学宫
- 殿才仙人：天仙；黑白学宫；封锁星火殿保护纪宁命简不被少炎氏探查；与余薇共同应对无间门在安澶郡的威胁
- 木子朔：返虚地仙；傀儡宗师；道侣俞夏神魂消融后精神颓废，头发白了许多；在燕山等待纪宁归来
- 白水泽（白叔）：阵法宗师，元神圆满；在燕山等待纪宁归来
- 小青（空青蛇）：乾坤大道域境；在燕山等待纪宁归来
- 秋叶：凡人侍女；在明心岛每日打扫纪宁书房等待归来；知道纪宁身处绝境
- 通合剑仙（绝剑仙）：近乎纯阳真仙实力；斩杀安澶北山氏近半返虚叛徒；与无间门在安澶郡对抗九年
- 吕洞宾（吕祖）：纯阳真仙极致，一只脚迈入道祖门槛；余薇之师

## 2. 关键法宝

- 摘星府（水府）：先天灵宝，内含女娲娘娘开辟的"摘星大世界"，阵法禁制由女娲布置，真神道祖也需靠实力强破；黄毛大熊（已修炼为真正生灵，未脱离摘星府）为府灵
- 千牛剑：仙阶极品之极品，近乎纯阳法宝之威，内含万里宿河
- 罗T弓：先天上品灵宝（上古十大神弓之一），弓弦已补齐（圆满状态），配数百根仙阶上品箭矢
- 三柄北冥剑：凶厉之气加持，媲美仙阶上品飞剑

## 3. 摘星府七天神（三寿道人麾下，卷末正式登场）

- 红雪天神（红发）：领袖，三寿麾下最强战将，天神最顶端，仅差一步成真神道祖；创《红雪》枪法
- 元光天神（稚童）：三寿道人之义子，跟随三寿最早，掌控摘星大世界大阵三人之首
- 雪蝎天神（妖娆白眉女子）：真性情，掌控摘星大世界大阵三人之一
- 九牙天神（秃顶老者）：最阴险，可化为黑色乌鸦
- 鸠蛇天神（温和少年）：上古剧毒神兽鸠蛇所化
- 烈阳天神（皮肤泛红魁梧男子）：烈阳真火中诞生的双胞胎神魔之一
- 玄月天神（皮肤泛黑魁梧男子）：玄月真水中诞生的双胞胎神魔之一
- 七人可合体施展"七曜天神阵"，化为七曜天神（以红雪为核心），达道祖门槛战力
