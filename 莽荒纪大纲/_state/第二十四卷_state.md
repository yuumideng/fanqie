# 第二十四卷 雾岩星主 · 滚动状态档案

## 1. 人物境界线（卷末）

- 纪宁本尊（化名"北冥"/"旭日剑仙"）：祖神/祖仙；半步世界神神体+青花雾气之力（力量速度达世界神门槛）；九重混沌禁制合一，识海凝聚"青花"，青花空间可转化三力为雾气力量；纯阳金丹第二等；摘星手第七转（借鉴《雾岩毁灭篇》+《九元灭》创出）；剑力第四层次"剑心"，无名剑术第一式"心剑式"大成，第二式"杀剑式"已悟透；《伍宝剑术》大成；神剑紫光琼完全修复（永恒神兵，引动一丝神剑本源威能）；获《雾岩毁灭篇》上篇、《黑雾剑术》玉简；雾岩军第六位将军+巡守者；随破烂衣袍道君前往天苍宫
- 纪宁本尊分身：在荒凉星辰世界牢狱中，亦九重禁制合一凝聚青花
- 纪宁第二元神：在混沌仙府镇守三界，亦九重禁制合一+青花雾气力量（世界境大能层次实力）
- 雾岩星主（游子辰）：世界神圆满境界（施展《雾岩毁灭篇》第三重）；传承之宝"雾岩星"；道侣"仪姐"被飞枭仙人所杀，本卷复仇成功击杀飞枭仙人；二叔黑雾世界神战死
- 黑雾世界神：施展《雾岩毁灭篇》第四重（残缺禁术，不可逆雾岩化）挡住黑莲神帝，剑术突破至防守境界，最终雾岩化身死，化作雕像
- 黑莲神帝（黑莲道人）：世界神巅峰（永恒神兵弯刀+黑莲帝都道之神兵+黑莲禁术）；师承七皇道君；被破烂衣袍道君一指击杀
- 飞枭混沌仙人：大罪孽祭塔（道之神兵），屠戮黎寒混沌世界杀雾岩星主道侣，被雾岩星主三剑击杀
- 心神将（源老人）：黑莲神帝麾下九神将之一，被纪宁以神剑紫光琼拦腰斩断斩杀
- 火仙子苏尤姬：纪宁第一个追随者，炼气流祖仙，媲美巅峰祖神，有望成世界境
- 破烂衣袍道君（天苍宫成员）：生死道君，称纪宁为"兄弟"，因纪宁持北休世界神接引令牌而认出，一指击杀黑莲神帝，带纪宁前往天苍宫
