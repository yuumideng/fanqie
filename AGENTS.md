# 项目协作入口

这是一个大型小说创作与原作结构研究项目。任何 Codex、Agent 或人工协作者开始工作前，必须先阅读：

1. `项目执行规则.md`：全项目通用规则；
2. `README.md`：项目结构、当前进度和事实口径；
3. `CHANGELOG.md`：近期变更和交接记录；
4. 与任务直接相关的总纲、索引、state 和任务日志；
5. 若处理任何一部原作的逐章分析，再阅读 `原作分析规则.md`。

入口职责仅限于说明“开始工作前读取什么”和“如何完成交付”，不在这里重复当前卷数、创作设定或事实边界。具体规则以 `项目执行规则.md`、`原作分析规则.md` 和当前总纲为准。

提交项目时，必须遵守 `项目执行规则.md` 中的完整 Git 流程：新建 `codex/YYYYMMDD-XX` 分支，更新 `README.md` 与 `CHANGELOG.md`，提交并推送工作分支，成功后合并回 `main` 并推送 `main`。
