# 第九集 妖乱人间 · 滚动状态档案

## 1. 人物境界线（本集结尾值）
- 孟川本尊：大日境神魔（圆满雷霆灭世魔体+刀魂巅峰），元神三层（诛神刺练成，元神兵器可凝练），肉身秘术"神通境"修炼中（已换赤血神丹），金刚之身已修成。速度封王神魔顶尖水准。单独执行巡查任务，任务范围扩至以顾山府为中心周围一千里。
- 柳七月：大日境神魔（凤凰神体+神箭手），有孕（双胞胎，九月初三产下一子一女）。获秦五尊者赐凤凰羽毛（最后一根），融入体内后凤凰血脉精纯，成封王神魔概率大增。
- 孟大江：不灭境神魔（炼体一脉），东宁府地网最高负责人，孟氏族长。
- 柳夜白：神魔（灭妖会），与孟大江联手负责东宁境内追查妖王妖族。
- 秦五尊者：造化境，元初山师尊，密切关注孟川元神天赋，赐凤凰羽毛，安排孟川独立任务。
- 洛棠尊者：造化境，驻洛棠关（大周唯一超大型城关），与秦五尊者共同关注孟川。
- 吕越王：封王神魔，驻洛棠关，曾赶来救援孟川（未及），见证孟川越阶杀四重天妖王。

## 2. 妖族境界线
- 伏角大妖王：四重天妖王（吴州唯一），被孟川越阶斩杀（本集核心战斗）。神通：斩敌金光、蚀灭气。肉身内外如一极强。
- 白狐妖王：三重天妖王（妖圣血脉后裔），神通魅惑，被孟川斩杀。
- 青蛇女妖（青玉妖王）：三重天妖王（道之境），擅江水领域，逃遁。
- 八臂妖王阐杜：三重天妖王，擅土遁，二月初一攻打府城时被孟川威慑逃遁。
- 猫妖王：三重天妖王（神通），第三只眼针对头脑，被孟川夫妻联手斩杀。
- 瘴游大妖王：原州新晋四重天妖王（本集提及，未出场）。

- [本集] 人族神魔中有叛徒勾结天妖门/妖族 —— 秦五尊者提及，待后续展开。

## 4. 核心主线进度
- 主线A：人妖战争线 —— 妖族大规模入侵人族世界（中小型城关全放弃），三大宗派组建"地网"+神魔小队+救援小队层层防御。孟川夫妇隐居顾山府任巡查，从府内救援到跨府救援，最终孟川单独执行一千里范围任务。
- 主线B：孟川修行线 —— 元神二层→元神三层（诛神刺练成），肉身秘术"神通境"修炼中，斩妖刀蜕变。越阶斩杀四重天妖王伏角大妖王。
- 主线C：孟川家庭线 —— 柳七月怀孕（双胞胎），九月初三产下一子一女。获凤凰羽毛，血脉精纯。

## 5. 近章摘要队列（最近10章）
- 第十七章：孟川给怀孕妻子绘画时元神突破至三层（九尺），夜游出窍经风劫，开始修炼诛神刺。
- 第十八章：诛神刺练成（七天凝练神纹），换三百九十枚赤血神丹修炼肉身神通境，元神三层之力全方位提升实力至封侯门槛。
- 第十九章：秦五尊者与元初山主商议妖族拟派三成三重天妖王入侵；东宁府安平县七妖王围攻神魔小队，伏角大妖王设伏等大鱼。
- 第二十章：孟川夫妇赶赴东宁府救援，伏角大妖王发现是孟川柳七月大喜，近距离突袭施斩敌金光。
- 第二十一章：孟川元神领域先发现伏角，挡下金光；双方强强碰撞，孟川以心刀式伤伏角，伏角施蚀灭气，孟川出诛神刺重创伏角元神。
- 第二十二章：孟川追杀伏角，龙吟式贯穿其身体多次（含心脏），伏角钻地逃，感叹孟川元神天赋。吕越王从洛棠关赶来救援。
- 第二十三章：孟川龙吟式刺穿伏角头颅将其斩杀，斩妖刀吞吸血气蜕变泛紫光。获储物布袋等战利品。吕越王赶到震惊，命保密。
- 第二十四章：吕越王掩护孟川（对外称自己杀的），回报洛棠关。秦五尊者与洛棠尊者推测孟川已达元神三层。
- 第二十五章：秦五尊者召孟川夜归元初山，确认元神三层。赐十方寂灭寒煞、安排独立任务（一千里范围）。战利品换功劳，发现幽玉瓶。
- 第二十六章：秦五尊者以凤凰羽毛换幽玉瓶给孟川（最后一根凤凰羽毛）。柳七月融入羽毛血脉精纯。九月初三柳七月产下一子一女，本集终。
