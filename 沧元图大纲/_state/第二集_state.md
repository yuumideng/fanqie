# 第二集 众生相 · 滚动状态档案

> 本档案由对应篇/卷/集现有大纲生成，用于后续 Agent 轻量接手；未对原文进行全库正确性审计。

## 1. 阶段概况
- **篇名**：众生相
- **集号**：第二集
- **章节数**：25章
- **时间跨度**：孟川踏入脱胎境约一年后（夏日）起，经历闲石苑天妖门事件、画《众生相》长卷（历时四个月余，至秋叶飘落）、悟出刀势、碧云楼斩血云盗、获得黑铁天书残片传承（两月余恢复）、二月初六夜天妖门三高手截杀之战，至次年三月二十五晏烬悟出剑势、三月底孟川公开悟出刀势止，合计约一年有余
- **空间跨度**：东宁府城（镜湖道院、云江酒楼、周府、闲石苑、碧云楼、漕运河、镜湖孟府、孟家祖宅、黑狼帮总部、天妖门东宁分堂地下据点等）

## 2. 核心人物与设定
**孟川至亲与师门**
- 孟大江：孟川之父，无漏境，悟出刀势，时常外出。本集中因孟川受伤后怕而震怒周家，后收千年人参赔礼。
- 柳夜白：柳七月之父，无漏境，悟出势（气息虚无缥缈），时常外出。与孟大江共同护卫孟川。
- 孟仙姑：孟家神魔，成神魔约八十年，身受重伤仅剩六七年寿命。擅探查（一念十里）、领域手段。本集中驱逐孟川体内妖气、确认眉心空间猜测、嘱咐悟势保密、擒获慕容游、拔除天妖门东宁分堂。
- 柳七月：孟川青梅竹马，十五岁突破脱胎境，箭术卡在瓶颈未达合一境。本集中参与闲石苑事件、背孟川撤离、发求救烟花。

**玉阳宫相关人物**
- 晏烬：玉阳宫背景少年，天赋不亚于孟川，修炼冰火神体，擅双剑。嫉恶如仇但性情冷漠。本集中闲石苑事件与孟川并肩作战受重伤，本集结尾悟出剑势。
- 老仆（王伯）：晏烬的仆从，无漏境实力，持玉阳宫令牌。闲石苑事件中服用神血丹拼命缠住天妖门副堂主高老三。后更换为更强的神秘强者暗中护卫晏烬。

**天妖门东宁分堂**
- 慕容游：东宁分堂堂主，白眉金瞳，凝练妖丹，神魔之下近乎无敌。为夺取黑铁天书残片策划截杀孟川，被孟川三刀重伤断臂后被孟仙姑擒获，审问后处死。
- 高老三（驼背男子）：东宁分堂副堂主，妖气外放，绿色雾气，擅指甲飞射（灭绝指）。闲石苑地下殿厅修炼妖法者，后被孟川一刀斩为两段。
- 屠尚（雄壮大汉）：东宁分堂副堂主，黑色妖气，擅大斧。被孟川借慕容游之刀引导误伤后以"摇落月"割喉斩杀。

**血云盗**
- 血手赵灿：血云盗二当家，无漏境，修炼"血魔手"（双手如蒲扇）。在东宁府覃县杀死三百余人后逃脱加入血云盗。被孟川断一手一腿后斩杀。
- 肥胖大汉：血云盗盗匪，被孟川在碧云楼雅间一刀斩杀。

**黑狼帮**
- 刘昶：黑狼帮帮主，无漏境。早知闲石苑有异却视而不见，事发后灭口知情弟子阿全。被判服兵役三年（沁阳关敢死军）。
- 华袍老者：闲石苑护法，脱胎境。

**周家**
- 周鹤：周潜之父，东宁府富商，颇有手腕。事发后带子请罪，以千年人参赔礼，借卷宗侧重点引导令白家忌惮孟家而不敢报复周家。
- 周潜：镜湖道院弟子，孟川师弟。暗中指使铁生找孟川求助救红雨，被罚一百鞭子。
- 红雨：周潜贴身丫鬟，被黑狼帮掳走后获救。
- 铁生：红雨之弟，孩童，求孟川救助姐姐。

**其他**
- 云符安：云家子弟，与孟家解除婚约后希望孟川越没出息越好。本集末得知孟川悟出刀势后摔碎满书房瓷器。
- 残疾老人：沁阳关老兵，断一腿一臂，两万军士仅一千六百三十三人生还。每日在河边看街上人们便觉开心，点醒孟川对众生的感悟。

## 3. 核心主线进度
本集以"众生相"为题，围绕孟川在东宁府的修行成长、心性蜕变与天妖门东宁分堂的覆灭两条主线交织展开，同时穿插黑狼帮事件、血云盗斩杀、黑铁天书残片获取等重要情节，最终以天妖门东宁分堂被一锅端、孟川悟出刀势公开收束全篇：

**闲石苑事件与众生相线**：孟川在镜湖道院指点师弟周潜剑法后，因周潜暗中指派孩童铁生求助，与玉阳宫少年晏烬联手前往黑狼帮据点闲石苑营救被掳女子红雨。在闲石苑发现被强迫调教的女子们各有辛酸故事，并非全是被逼迫，令孟川初识世间百态。搜查过程中惊动潜伏于此的天妖门东宁分堂副堂主高老三（驼背男子），其妖气外放出手袭击，孟川、晏烬重伤，老仆服用神血丹拼命缠住敌人才得以脱险。事后孟川翻看玉阳宫结案卷宗，深感世间种种无奈——救出的女子大多因债务等原因再度走向泥潭。街头所见残疾老人（沁阳关老兵）的豁达满足令孟川释然，悟出"人有千百种"的道理。孟川历时四个月余绘成巨幅长卷《众生相》，画出东宁府三教九流形形色色，心灵生出莫大圆满感，由此触发眉心空间（识海）觉醒，发现"心魂之力"。

**孟川修行蜕变线**：眉心空间觉醒后，孟川获得闭眼感应周围十丈纤毫毕现、一里内感应一切生命气息的能力，并将此未知力量命名为"心魂之力"——可融入肉身或刀法中短暂暴增实力。孟川拜访父亲与姑祖母孟仙姑询问眉心空间，孟仙姑猜测此即传说中的"识海"，内有魂魄，关系到更强大神魔的修炼，嘱咐保密待入元初山查阅典籍。孟川在修炼中发现心魂之力的妙用，并在一夜苦修中触摸到"势"的门槛。随后在融入心魂之力施展极限拔刀式时，仅第四刀便悟出刀势（十六岁），震惊父亲孟大江与柳夜白。孟仙姑嘱咐保密至次年年中再公开。

**血云盗与黑铁天书残片线**：孟川在碧云楼凭感应能力辨认出血云盗二当家"血手赵灿"的易容身份，当场斩杀另一名大盗并追杀赵灿，断其一手一腿后击杀。从赵灿遗物中获得残破黑铁片，意识被吸引其中接受传承——黑铁天书《雷霆灭世刀》第十七式（仅前两刀诀窍），昏迷受伤两月余方恢复。天妖门堂主慕容游出关后辨认出黑铁片为黑铁天书残片（碎裂为26块之一，价值一万大功），决定活捉孟川逼问残片下落。

**天妖门覆灭线**：二月初六夜，天妖门东宁分堂堂主慕容游率两位副堂主（高老三、屠尚）截杀孟川。孟川凭借十丈感应领域与心魂之力，假装仓皇逃命引诱敌人追近，以"雷霆极限拔刀式"一刀将高老三斩为两段；继而利用十丈领域精准掌控战场，借慕容游之刀引导误伤屠尚，以阴柔刀法"摇落月"割喉斩杀屠尚；最后追杀慕容游至漕运河边，施展自创三刀组合（五绝灭世前两刀+摇落月）重伤慕容游断其一臂，但心魂之力耗尽未能击杀，慕容游跳入漕运河逃遁。孟仙姑赶到，以神魔手段从河底擒获慕容游。审问后处死，连夜拔除天妖门东宁分堂全部据点，东宁分堂覆灭。

**收束**：次年三月二十五，晏烬悟出剑势消息传出；隔三天孟家公开孟川悟出刀势消息，十七岁悟出势令五大神魔家族震动，云家云符安摔碎满书房瓷器。第二集至此完整收官。

## 4. 近章摘要队列
### 第十六章 血云盗二当家
**类型标签**：悬念设置、伏笔深化
**内容**：孟川进入碧云楼引发青楼女子们轰动。孟川在雅间借助十丈感应，清晰"听到"同在二楼另一雅间的血云盗两名大盗对话——大胡子男子易容，计划再玩两天离开东宁府进山。孟川凭感应辨认出大胡子的易容手段（假发假胡子假皮），联想到通缉犯"血手赵灿"（光头精瘦、左脸黑痣），确认其身份为血云盗二当家。孟川起身前往二盗所在雅间。
**核心**：孟川凭感应识破血手赵灿易容身份，确认血云盗二当家行踪。

### 第十七章 刀斩血云盗
**类型标签**：战斗描写、转折高潮
**内容**：孟川进入雅间当场一刀斩杀肥胖大汉。血手赵灿（无漏境）从窗户逃出，孟川追杀。碧云楼客人们目睹孟川杀意冲天。赵灿施展"血魔手"双掌抵挡，被孟川断其一臂后施爆发神魔禁术撞墙逃遁。孟川追出一里地，施展出五成实力并融入心魂之力，断赵灿一腿。赵灿以银票、黑铁片和血云盗宝藏求饶，孟川不为所动，一刀割喉将其斩杀。孟川收走银票和黑铁片。
**核心**：孟川斩杀血云盗二当家血手赵灿，获得黑铁天书残片。

### 第十八章 传遍全城
**类型标签**：情节推进、人物心理
**内容**：孟川吹哨召集孟家人处理尸体。斩杀血云盗二当家的消息传遍全城，云符安等富家子弟议论纷纷，推测孟川达合一境巅峰、两三年内可能悟出势。孟大江将一万六千两战利品银票交给孟川。孟川夜间研究黑铁片，意识被吸引其中。
**核心**：孟川斩杀血云盗消息传遍全城，外界低估其实力；孟川意识被黑铁片吸引。

### 第十九章 《雷霆灭世刀》
**类型标签**：设定展开、伏笔深化
**内容**：孟川意识被引入黑铁片中，看到一名伟岸男子在山巅挥劈一刀，恐怖刀光劈中孟川意识致其昏迷头痛流血。醒来后头痛数日方缓，眉心空间小人黯淡许多。三天后开始研究传承讯息——为《雷霆灭世刀》第十七式"五雷降世"（注：后文第二十三章称"五绝灭世"），仅得前两刀诀窍（共五刀），每一刀比前一刀更强，第五刀产生质变。两个月余完全恢复后开始修炼前两刀，需调动心魂之力方能施展。同时天妖门堂主慕容游出关，屠副堂主告知黑铁片之事。
**核心**：孟川接受黑铁天书《雷霆灭世刀》传承，仅获第十七式前两刀诀窍，天妖门堂主慕容游出关。

### 第二十章 黑铁天书残片
**类型标签**：设定揭示、悬念设置
**内容**：慕容游翻查画卷，确认血云盗所卖黑铁片为黑铁天书《雷霆灭世刀》残片（完整天书碎裂为26块之一）。完整黑铁天书贡献妖族可换一百万大功，一片残片值一万大功。三位天妖门首领决定不惜一切夺取残片。天妖门以妖术控制孟川小院打扫的老仆和丫鬟搜查，未找到黑铁片（孟川将其藏在父亲地下密道密室中）。慕容游决定活捉孟川逼问残片下落。
**核心**：黑铁片身份确认为黑铁天书残片，天妖门决定活捉孟川逼问。

### 第二十一章 夜幕下的截杀
**类型标签**：矛盾爆发、战斗描写
**内容**：二月初六夜，孟川与柳七月、晏烬饭后分别。孟川感应到三股妖异气息逼近，让柳七月先走。天妖门三高手——堂主慕容游（金瞳白眉，凝练妖丹）、副堂主高老三（驼背男子）、副堂主屠尚（雄壮大汉）——三方围堵截杀孟川。孟川假装仓皇逃命，拉动求救烟花筒。孟川选择向高老三方向突围，待其追近后瞬间拔刀，施展"雷霆极限拔刀式"（心魂之力+十五倍真气+刀势+雷霆灭世刀诀窍），一刀将高老三斩为两段。
**核心**：天妖门三高手截杀孟川，孟川以雷霆极限拔刀式一刀斩杀副堂主高老三。

### 第二十二章 什么怪物!!!
**类型标签**：战斗描写、转折高潮
**内容**：慕容游、屠尚震惊于孟川一刀斩杀高老三。孟川补刀杀死仅剩上半身的高老三后，转而杀向屠尚。屠尚心惧逃向慕容游方向。孟川以十丈领域精准掌控战场，利用慕容游挥出的弯刀，以刀轻碰引导其变向误伤屠尚，趁乱以阴柔刀法"摇落月"割喉斩杀屠尚。慕容游不敢相信成了帮手。孟川目光转向慕容游。
**核心**：孟川借力打力以慕容游之刀误伤屠尚，趁机斩杀副堂主屠尚。

### 第二十三章 孟川的第三刀
**类型标签**：战斗描写、转折高潮
**内容**：慕容游逃向漕运河。孟川正常速度跟随（不动用心魂之力），等待姑祖母赶到。慕容游逃至河边前，孟川决定最后一搏，心魂之力融入全身，施展黑铁天书《雷霆灭世刀》第十七式"五绝灭世"——第一刀霸道堂皇，第二刀更霸道令慕容游脏腑受震，第三刀（无诀窍）将前两刀蓄积的雷霆真气全部爆发，以苦修已久的"摇落月"阴柔刀法施展，温柔邪异轨迹难捉。慕容游挡住前两刀，第三刀被旋转诡异力道摆脱弯刀阻碍，断其一臂并重创胸口。但孟川心魂之力近乎枯竭，慕容游施展禁术跳入漕运河逃遁。
**核心**：孟川施展自创三刀组合（五绝灭世前两刀+摇落月）重伤慕容游断臂，但心魂之力耗尽未能击杀。

### 第二十四章 慕容游的结局
**类型标签**：转折高潮、矛盾收束
**内容**：慕容游跳入漕运河潜深逃遁，孟川感应不到一丈深以下的气息。孟仙姑赶到，以拐杖碰触河面释放无形波动，丝线缠绕擒获慕容游从河底拖出。孟仙姑此前已目睹孟川爆发实力追杀慕容游的全过程，震惊于孟川之强。孟川恭敬行礼。
**核心**：孟仙姑以神魔手段从漕运河底擒获重伤的慕容游，截杀之战收束。

### 第二十五章 天下事（本集终）
**类型标签**：矛盾收束、设定展开
**内容**：孟仙姑封禁慕容游感官后与孟川交谈，得知孟川爆发实力源于眉心空间的心魂之力，嘱咐继续保密。孟仙姑审问慕容游，以折磨手段迫使其交代天妖门东宁分堂全部情报后处死。当夜孟仙姑率九位长老及一百二十位脱胎境高手分七路，一个时辰内拔除东宁分堂总部及六个据点，骨干一锅端，仅一名脱胎境逃脱。孟仙姑将天妖门三首领之死记在自己功劳上以保护孟川。孟川翻看缴获书籍，了解天下三分格局（大周王朝-元初山、黑沙王朝-黑沙洞天、大越王朝-两界岛）及妖族强大、天妖门为人族叛徒等天下大势。三月二十五晏烬悟出剑势消息传出，隔三天孟家公开孟川悟出刀势消息，十七岁悟出势令五大神魔家族震动，云家云符安摔碎满书房瓷器。
**核心**：天妖门东宁分堂被一锅端覆灭，孟川悟出刀势消息公开震动东宁府，第二集完整收官。

## 5. 伏笔与待确认项
**核心级**：
1. 眉心空间/识海/魂魄的真正奥秘：孟仙姑猜测关系到更强大神魔的修炼，具体内容待入元初山翻看典籍揭晓。此为本集最重要的未解伏笔。
