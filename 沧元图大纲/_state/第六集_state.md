# 第六集 英雄征四方 · 滚动状态档案

## 1. 人物境界线（当前值）

- **孟川**：
  - 境界：大日境神魔（本集从丹云境→不灭境→大日境，九玄洞闯过下山）
  - 神魔体：雷霆灭世魔体·九炼圆满
  - 刀法境界：刀魂大成（本集从刀意巅峰→悟出刀魂→刀魂大成），已接受黑铁天书《心意刀》与《天地游龙刀》完整传承
  - 元神：元神第二层（本集从第一层突破至第二层，领域扩至二十丈，感应两里范围）
  - 核心装备：斩妖刀（天级神兵，原为神兵洞窟中两千年六百余年的邪刀，能吸血肉罪孽煞气，曾名"心魔刀""杀劫""无绝"）、深青色衣袍战靴护臂一套、贴身内甲
  - 当前所在：下山途中，飞往北河关（距东宁府一万三千里）
  - 特殊传承：沧元洞天所得"肉身修行体系"（筑基/金刚/神通/不死/滴血/入圣/神话七重，需星空晶石，在人族世界最高可修至滴血境即媲美封王神魔；已得微小星空晶石开辟中丹田）
  - 元神之力效果：融入大日真元实力提升一倍（可施五十招）；仅调动肉身提升六成（可施十五招）；全力爆发实力达三倍（仅七招）

- **柳七月**：
  - 境界：大日境神魔（本集突破，与孟川一同下山）
  - 神魔体：凤凰神体（魂之境大成后可凝练凤凰羽翼飞行）
  - 弓箭/火焰境界：魂之境大成（实际一个月前已达到，故意隐瞒，等待孟川）
  - 修炼方向：天级神魔绝学（凤凰火焰一道），有完整意境引领
  - 核心装备：天级神兵（神兵洞窟所得，弓箭类）、青红色衣袍
  - 当前所在：与孟川一同下山飞往北河关
  - 与孟川关系：青梅竹马，私定终身，约定下山后成亲

- **晏烬（安海王家七公子）**：
  - 境界：不灭境神魔（青莲神体大成，未圆满）
  - 修炼：黑铁天书《冰火七绝》（需悟冰剑意+火剑意，十一年仅悟出冰剑意，无法入门）
  - 性格：偏执孤僻，居住飘雪峰，朋友仅孟川、柳七月
  - 当前所在：元初山飘雪峰

- **阎赤桐**：
  - 境界：不灭境神魔（青莲神体圆满，十六岁成神魔）
  - 修炼：黑铁天书《红莲火焰枪》（已练成）
  - 身份：被特别邀请上山的少年天才，十三岁悟势
  - 当前所在：元初山（未下山）

- **姬元通**：
  - 境界：神魔（大力魔体大成后闯生死关成神魔）
  - 经历：曾消失三年，再无傲气，低调朴素
  - 当前所在：元初山

- **钱钰**：
  - 境界：大日境神魔（丹田彻底崩溃，修行路断绝，成废人）
  - 经历：槐山关被攻破，二十六位神魔战死仅其独活，施展禁术杀六十二位妖王
  - 当前状态：颓废疯癫，终日饮酒
  - 当前所在：元初山洞府

- **尊者（秦五，师尊）**：
  - 境界：造化境（超越封王神魔）
  - 身份：元初山近三百年所有弟子共同师尊
  - 本集事件：赐孟川两部黑铁天书传承、为孟川开启沧元洞天（联合洛棠尊者）

- **洛棠尊者**：
  - 境界：造化境
  - 身份：镇守洛棠关（大周王朝七大城关之首）
  - 本集事件：与秦五尊者共同开启沧元洞天

- **元初山主（闻攸）**：封王神魔，本集亲自为战死弟子刻名于赤血崖
- **易长老**：封王神魔，赐孟川柳七月神魔法门下册及元神秘术，安排下山分配
- **南云侯**：封侯神魔，护送孟川等五名弟子前往北河关
