# 第四集 元初山 · 滚动状态档案

## 1. 人物境界线（当前值）
- 孟川：凡俗凝丹高手，刀势巅峰（离刀意一步之遥），修炼雷霆神体，眉心空间内有"心魂"（疑似天生元神，未公开），速度超绝（初选第二项七滴水，全场第一），灵觉极高（初选第三项闯过第十轮、败于第十一轮，评价"超品"，数百年一出），已正式拜入元初山内门，入门考核排名第一，选定修炼超品法门"雷霆灭世魔体"，当前位于景明峰洞府
- 晏烬：凡俗凝丹高手，冰火神体，剑术境界稚嫩（初选第一项一百三十七层、第二项十二滴），入门考核初选第九、终选前五，选定修炼超品法门"青莲神体"，洞府在飘雪峰
- 姬元通：凡俗凝丹高手，力量型天赋极强（初选第一项两百九十九层、全场第一），灵觉仅次于孟川（初选第三项闯过第七轮），但内心意志薄弱（黑暗祭坛仅第十七层，百人中倒数第九），入门考核排名第十五，选定修炼超品法门"大力魔体"，被元初山下令立即下山前往潼阳关敢死军服兵役一年以磨砺内心意志
- 宗沙：凡俗凝丹高手，风之神体，神箭手（初选第一项六十五层、第二项九滴、第三项闯过第七轮），入门考核排名第二，选定神魔法门未明确交代
- 楚雍：凡俗凝丹高手，水之神体兼修少许雷霆，王都第一天才（初选第一项一百九十一层、第二项十二滴、黑暗祭坛三十八层），入门考核排名第七，选定修炼超品法门"沧海魔体"
- 孟大江：孟川之父，凝丹高手，已返回东宁府孟家
- 孟仙姑（孟烟）：成神魔近八十年，外门弟子，不灭境神魔，年初妖族入侵重伤后身体崩溃，腊月二十三得知孟川入选元初山后当晚含笑而逝，临终将十八万两千两百功劳全部转给孟川
- 柳七月：已入元初山（凤凰神体血脉，直接召入），洞府在景明峰
- 易长老：封王神魔，元初山暮年强者，负责传法阁，亲自传孟川雷霆灭世魔体上册，暗示元神相关秘密

## 2. 伏笔追踪（已埋未解）
- [本集第1章] 孟川眉心空间内的"心魂"——疑似天生元神，连孟仙姑都未听说过，易长老暗示"天生元神"若被妖族知晓会遭不惜代价刺杀，孟川猜测眉心小人可能是元神，此秘密尚未公开 —— 待回收
- [本集第21章] 易长老对孟川说"最近三千年人族没听说有天生元神的"，并提到妖族也在探寻元神秘密，有意还是无意？—— 待回收
- [本集第16章] 姬元通内心意志薄弱，元初山为其制定多环节磨砺计划（第一环：潼阳关敢死军服兵役一年），后续环节尚未展开 —— 待回收
- [本集第18章] 赤血崖一万五千两百七十一位元初山神魔影像，其中一万一千九百六十八位战死沙场——元初山弟子守护人族的历史背景 —— 长线背景
- [本集第19章] 云符安（云青萍之父）对孟家解除婚约心怀不甘，表面讨好内心期盼孟家栽跟头—— 待回收

## 3. 核心主线进度
- 主线A：孟川元初山入门考核线 —— 从腊月二十一初选到正式拜入元初山内门，考核排名第一，选定雷霆灭世魔体，本集完整完成入门阶段
- 主线B：孟仙姑逝去线 —— 年初妖族入侵重伤后凭执念撑到得知孟川入选元初山，当晚含笑而逝，将毕生功劳转赠孟川，留下三条忠告，本集收束
- 主线C：各天才命运分线 —— 姬元通因内心意志薄弱被派往敢死军磨砺；楚雍选沧海魔体；晏烬选青莲神体；阎赤桐选两界神体；各人命运未定 —— 留待后续

## 4. 近章摘要队列（最近5-10章一句话）
- 第12章：东河王率众飞上元初山，终选第一项"斩妖"开始，红衣女子神魔释放一千二百二十七头妖怪
- 第13章：百名天才结圆桶阵斩妖，一炷香后仅孟川、晏烬、姬元通、楚雍、燕凤、宗沙、李英七人站立
- 第14章：终选第二项"黑暗祭坛"考验内心意志，姬元通在第十七层就停下（倒数第九），令众神魔震惊
- 第15章：孟川在黑暗祭坛幻象中愤怒自身软弱，意志凝练如刀，继续往上走
- 第16章：孟川闯至第七十五层（百人中第五），晏烬第八十一层，章烈第九十三层（第一）；东河王等商定二十人名单
- 第17章：最终名单公布，孟川第一，共二十一名弟子（含特别召入的阎赤桐）；孟大江返乡报喜
- 第18章：入门大典在赤血崖举行，山主主持，展示一万五千余位元初山神魔影像；新弟子点燃魂灯、选洞府、开始选神魔法门
- 第19章：新弟子在藏书洞阅读《神魔法门杂谈》，了解十二种超品、二十七种上品法门；孟川选定雷霆灭世魔体；孟仙姑当夜含笑而逝
- 第20章：孟川领取丹药宝物及功劳，得知孟仙姑转赠十八万两千两百功劳；孟仙姑家书寄达
- 第21章（本集终）：孟川读姑祖母遗书（三条忠告），正式接受雷霆灭世魔体上册传承，易长老暗示元神秘密；各天才各有命运选择
