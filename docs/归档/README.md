这里保存已经被当前版本替代、但仍有历史查阅价值的过程产物。归档文档不参与当前创作决策，不得覆盖 `项目执行规则.md`、`原作分析规则.md` 或当前总纲。

## 归档边界

- `*_旧版.md`：被后续分析报告或当前执行框架替代的研究、技术和状态文档；
- `旧版合并大纲/`：早期把多卷/多篇合并在单个文件中的大纲，当前以“一篇/一卷一个文件”的规范文件为准。
