## 一、项目定位

这是一个**网文写作结构逆向工程项目**。对网络作家"我吃西红柿"（番茄）的多部长篇代表作做**原文逐章解析**，提炼成结构化写作大纲，并横向对比叙事节奏，目的是拆解番茄的写作节拍规律。

核心产出物：每本书按篇/卷/集切分的**逐章大纲**，每章含「类型标签 + 80-150字忠于原文的概括 + 一句话核心」，辅以篇级整体主线梳理与滚动状态档案（人物境界/伏笔/主线进度）。

技术架构（见 `docs/吞噬星空后续大纲生成技术方案.md`）：
1. 脚本按章分割原文 → 单章文件落盘（可断点续传）
2. 单章读取 + 每篇轻量 `state.md` 滚动状态档案（替代全量原文上下文，控制上下文长度）
3. 篇级子 Agent 并行生成，主 Agent 管进度与探活
4. 篇级聚合产出大纲

铁律（`吞噬星空大纲/项目规则.md`）：忠于原文严禁臆造、存疑即查或问绝不猜测、章节不合并不拆分、数据引用索引。
