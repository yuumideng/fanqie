# 当前接手卡

> 这是项目导航卡，不是质量报告；只用于新 Agent 快速定位，不代表本次任务已验证全库。

- 项目：五部原作的逐章分析、大纲、state、汇总附录、特征数据和横向节奏研究。
- 正式作品：《吞噬星空》《莽荒纪》《飞剑问道》《沧元图》《雪鹰领主》。
- 接手顺序：先读 `AGENTS.md`，再读本卡；不要因为接手而打开 README、完整任务日志、整部原文或全量附录。
- 当前任务：以用户当前指令为准；本卡不保留临时任务，避免把旧待办误当成新任务。
- 修改章节：只读目标规则条款、目标大纲、目标 state 和目标原文；需要时再增加相邻章节。
- 结构/数量/完整性核查：只有用户明确要求，或当前修改必须依赖时才做。
- 大文件查询：优先使用 `tools/query_material.py` 定点查询，不直接打开全量附录、CSV 或长原文。
- 状态读取：先取 state 的快速视图；只有跨篇连续性或冲突核查才读完整 state。
- 任务结束：只记录完成、未完成、存疑和下一步目标文件，不把整段过程留在聊天上下文中。

详细规则按触发条件读取：项目执行规则、原作分析规则、Git 交付规则均不是接手阶段必读资料。
