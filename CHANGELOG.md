# Changelog

本文件记录项目每次正式提交的主要变更，按时间倒序排列。提交项目时必须同步更新本文件和 `README.md`。

## 2026-07-23

### 明确“提交”包含推送闭环

- 用户说“提交”即视为授权本地提交、推送工作分支、合并回 `main` 并推送 `main`；
- 仅在认证、网络、权限或工具实际失败时报告阻塞，不重复确认正常交付意图。

## 2026-07-23

### 完善上下文分层、定点查询与交接机制

- 分支：`codex/20260723-07`
- 新增 `docs/当前接手卡.md`，将新 Agent 的初始读取压缩为最小导航信息；
- 新增 `tools/query_material.py` 和 `docs/资料查询说明.md`，支持 Markdown、TXT、CSV 的按标题、关键词、作品和章节定点查询；

- 增加 state 快速视图、状态声明与事实核查分离、子 Agent 精简回报和任务收尾记录规则；
- 压缩 README 与任务日志中的重复规则说明。
