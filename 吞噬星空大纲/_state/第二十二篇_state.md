# 第二十二篇 银河领主 · 滚动状态档案

## 1. 人物境界线（当前值）

- 罗峰本尊（地球人）：宇宙之主（初入层次），称号"银河领主"；生命基因层次未达万倍（神力比其他宇宙之主差一筹）；核心装备：弑吴羽翼（四翼残缺至宝，第二形态"金色的国度"可掌控时空，半径80光年）、绝客（巅峰至宝战刀）、劫甲（巅峰铠甲至宝）、九虓河（顶级领域类至宝）、火焰沙杯（顶级领域类至宝，本篇新得）、星辰塔（至强至宝，仅能催发第二层）；驻扎银河系地球，永驻家乡。
- 金角巨兽分身：宇宙之主层次（神力补充地球人本尊）。
- 魔杀族分身：宇宙之主层次，速度型，本篇中用于夺取火焰沙杯。
- 幽海分身（无尽幽海）：神体海洋几百亿公里直径，类似九幽之主，可施展"我为宇宙"（微型宇宙）天赋秘法。
- 混沌城主：罗峰老师，原祖大弟子，虚拟宇宙一脉领头者，顶尖宇宙之主。
- 冰峰之主：原祖弟子，混沌城主师弟，顶尖宇宙之主，本篇与罗峰搭档参与垣奥秘境之战。
- 黑暗之主：原祖弟子，神秘顶尖宇宙之主，幽侯之主之师。
- 幽侯之主：黑暗之主弟子，宇宙之主。
- 龙行之主：虚拟宇宙公司元老，宇宙之主。
- 巨斧创始者：人类族群宇宙最强者，本篇中前往宇宙海倾峰界探索（持有倾峰界残图）。
- 原祖（"原"）：虚拟宇宙创始者，人类第一位宇宙最强者，被宇宙本源永久镇压，仍活着，雕像即其本体所在，虚拟宇宙本源即其本体。
- 垣奥之主：宇宙诞生早期崛起的特殊生命宇宙之主，独来独往，拥有奇物"黑色神木"（次等至强至宝），本篇末自爆殒落。
- 天狼之主、幽魔之主：妖族宇宙之主，持梦妖祖借予的"梦祖域"（巅峰领域类至宝）和"魔屿藤"（巅峰攻击至宝）。
- 焱帝：坐山客弟子（三师弟），九域联盟宇宙之主。
- 黑宙创始者：独行宇宙最强者，持至强至宝"黑宙神甲"，本篇夺走奇物"黑色神木"。
- 莫洛（狂兽侯）：封侯级不朽，南狐国主弟子，本篇被罗峰下令处死（因早年迫害罗峰之仇）。

## 4. 近章摘要队列（最近10章）

- 第20章：各方宇宙之主陆续赶到垣奥秘境，均被垣奥之主蹂躏击败，罗峰发现人类族群未到，回虚拟宇宙找混沌城主。
- 第21章：混沌城主派冰峰之主随罗峰前往；各方商议唯有"镇压封印"一法。
- 第22章：黑宙创始者现身，提议以奇物交换，各方同意；九大势力联手追杀垣奥之主进入空间夹层。
- 第23章：黑宙创始者压制垣奥之主，冰峰之主用巅峰镇封至宝"冰山殿"吞吸；垣奥之主绝望中亮出旋元遗留的倾峰界残图（牵丝图）。
- 第24章：各方确认牵丝图价值极高；冰峰之主、罗峰受混沌城主令务必夺取；垣奥之主撕裂牵丝图成数十份，各方疯狂争夺。
- 第25章：黑宙创始者取奇物神木；九大势力宇宙之主疯狂争夺牵丝图碎片；罗峰得第一份，焱帝愤恨。
- 第26章：罗峰与冰峰之主联手争两份大碎片，遭天狼之主、幽魔之主以梦祖域、魔屿藤两大巅峰至宝突袭，被迫放弃。
- 第27章：罗峰分头争夺，共得2份；冰峰之主得5份；人类共7份；星空巨兽联盟最多10份；黑宙创始者携奇物离去。
- 第28章：罗峰以微型宇宙+金色国度+九虓河三重叠加压制天狼之主，魔杀族分身夺取火焰沙杯（顶级领域类至宝）；罗峰、冰峰之主回归人类疆域。
- 第29章（终章）：罗峰向混沌城主回禀全程；混沌城主得知弑吴羽翼乃残缺至宝（巅峰至宝极品部件）大惊；准许罗峰进入宇宙海；终章揭示三位大破灭前古老存在布下大局。
