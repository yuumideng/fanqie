# 第二十五篇 烽火边疆 · 滚动状态档案（篇末最终态）

## 1. 人物境界线（当前值）

- 罗峰本尊（地球人）：
  - 境界：宇宙之主（本篇第37章突破，原为宇宙尊者；第36章观看"掌上宇宙"感悟灵光突破瓶颈；突破后四大分身均达六阶顶尖战力）
  - 意志：宇宙最强者层次
  - 核心装备：星辰塔（至强至宝，本篇揭示其镇守灵魂效果超乎寻常）、疯魔灭神甲、弑吴羽翼（前两对，可施展金色国度）、血影刀、墓陵之舟、传承空间、雪沙海（巅峰领域至宝）、巅峰灵魂类至宝、古神眼×2（本篇第21、32章由坐山客用两滴神血炼制，意志冲击可达宇宙最强者层次）
  - 秘法造诣：三大刀法《雾光虚空》《金之宇宙》《涅槃新生》；断东河一脉三大绝学之一《断灭》（令神力加速燃烧提升一层次）
  - 神体：无尽幽海（直径约800亿公里，突破后转化约15%为万倍宇宙之主级神力；可扩张至最大直径3000亿公里；移动艰难，需借神国通道转移至星辰塔内）
  - 身份：断东河一脉第193任传承者；人类族群四大支柱之一
  - 当前所在：原始秘境（本篇终章）

- 金角巨兽分身：宇宙之主（本篇第37章突破），携弑吴羽翼（白色羽翼部分）、星辰塔、劫甲；可施展《断灭》爆发六阶顶尖战力；本篇第27章星辰塔首次显现吞吸妖族舰船

- 魔杀族分身：宇宙之主（本篇第37章突破），生命基因层次提至3000多倍；本篇持古神眼参战捕获宇宙之主

- 幽海分身：宇宙之主神力，1000倍基因层次；新幽海分身千倍基因

- 混沌城主（罗峰老师）：宇宙之主五阶顶级；本篇揭示原祖所留"掌上宇宙"（机械流宝物，影响范围1光年，可同时施展两次六阶顶尖攻击，需两大分身操控）；本篇终章宣布人类进入"强盛时代"

- 黑暗之主：宇宙之主五阶顶级；持黑狱塔（至强至宝宫殿，施"黑暗沉沦"）；本篇终章与罗峰共同捕获208名宇宙之主

- 彭工之主：宇宙之主五阶顶级；持延钧棍（至强至宝攻击类，延伸成甲媲美疯魔灭神甲）

- 原祖：被永久镇压，虚拟宇宙创造者；本篇揭示虚拟宇宙覆盖整个原始宇宙可作战场监控；本篇第4章现身决定至宝分配；留下"掌上宇宙"镇族之宝

- 巨斧创始者：宇宙最强者，本篇开篇已殒落（承接第24篇末），小型宇宙崩溃

- 坐山客（罗峰老师）：独行宇宙最强者，本篇拒绝加入任何阵营；第20、32章为罗峰炼制两颗古神眼（用两滴神血）；揭示其与罗峰师徒关系被星空始祖外泄

- 东帝始祖：宇宙海最古老存在；本篇交易僵持，索要第二颗记忆之石被拒；本篇终章对人类获胜不满

- 紫月始祖：宇宙海最古老存在（超脱轮回者）；本篇第46章揭示其暗中给予自由联盟十倍详细传承，意图对付罗峰；本篇终章将传承传授联军各族以遏制人类

- 梦妖祖/震妖祖：妖族宇宙最强者；本篇第一战损失6位宇宙之主，最终战败退

- 虫族女皇：虫族宇宙最强者，持女皇宫（至强至宝宫殿）；最终战靠女皇宫带联军阵营11位最强者逃出无尽幽海

- 机械族父神：机械族宇宙最强者，操控10081座机械傀儡（群海战术）

- 魔祖（狱族）：狱族宇宙最强者，本篇散播巨斧殒落消息

- 始化岛主：独行宇宙最强者，自由联盟盟主；本篇建立自由联盟（298强大族群、433名宇宙之主）；因紫月圣地介入失去掌控权，最终战中立不出手

- 光行之主：本篇第45章被紫月圣地推举为自由联盟实际代表，取代始化岛主

- 三大祖神：祖神教领袖，实为宇宙之主（非宇宙最强者），原始宇宙本源赐予言出法随特权与天赋秘法；持五彩极光湖（领域至强至宝）、祖神坛；本篇加入人类阵营

- 星空始祖（老兽神）：星空巨兽联盟领袖，宇宙最强者；本篇加入联军阵营；外泄坐山客与罗峰师徒关系

- 北疆盟主：独行宇宙最强者，持栖曲星光湖（领域至强至宝）；本篇加入联军阵营

## 3. 核心主线进度

- 主线A：巨斧殒落后人类备战与至宝分配线 —— 巨斧小型宇宙崩溃→抢救精英→混沌城主定方针→罗峰与东帝圣地完成第一次交易获黑狱塔、延钧棍→原祖决定分配（黑暗取黑狱塔、彭工取延钧棍、混沌坐镇）→坐山客炼制古神眼×2→人类组三支队伍。**完成**
- 主线B：第一次族群大战（对妖族）线 —— 三族出征→人类三队联合伏击妖族梦妖祖队→示弱后反攻→血影刀首次出鞘斩杀1名妖族宇宙之主→墓陵之舟+神血意志冲击捕获5名（3名有分身自爆）→妖族共损失6位宇宙之主→谈判破裂。**完成**
- 主线C：第二次族群大战（对三族联军）线 —— 三族联军112名宇宙之主出征→古神眼扫荡+黑暗沉沦+领域束缚→星辰塔首次显现吞吸妖族舰船16名宇宙之主→三族大败，损失十余位→谈判再次破裂。**完成**
- 主线D：势力拉拢与自由联盟建立线 —— 六大势力会盟→双方拉拢闲散族群→始化岛主建立自由联盟（433名宇宙之主）→星空巨兽联盟、祖神教提条件→罗峰突破宇宙之主→人类答复（仅给成宇宙最强者指引）→始化岛主私下交易→紫月圣地介入。**完成**
